# TUI overlay aliases during pending turns

The TUI resolves slash-command aliases before deciding whether an input should
open a local overlay or queue behind an active provider turn.

Compatibility contract:

- `/settings` opens the same Config text overlay as `/config`.
- `/bashes` opens the same Background Tasks picker as `/tasks`.
- `/checkpoint` opens the same Rewind picker as `/rewind`.
- Read-only local overlays, including `/config`, may open while a provider turn
  is pending.
- Mutating commands selected from those overlays, or entered directly as aliases
  such as `/bashes stop <id>` and `/checkpoint`, still queue until the pending
  provider turn settles.

Regression coverage lives in `crates/puffer-tui/src/tests/panels.rs`.
