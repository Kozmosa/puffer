# Connect Search Connection Names

The TUI slash popup now mirrors the deterministic core `/connect` parser when a
connector is selected through search terms.

- Two-token `/connect <connector-search> <connection-name>` popup rows preserve
  the typed connection name in the replacement command.
- The popup still uses all typed terms first so metadata searches such as
  `telegram personal` disambiguate connector rows before preserving `personal`
  as the connection name.
- If the full query does not match, the popup falls back to matching only the
  connector-search token so custom names such as `matrix-main` still complete to
  `/connect matrix-bot matrix-main`.
- Exact `/connect <connector-slug> <connection-name>` inputs remain complete
  commands and do not render the popup.
