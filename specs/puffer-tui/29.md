# AskUserQuestion focusable Other row

The TUI `AskUserQuestion` overlay treats the rendered `Other` row as a real
keyboard target.

Compatibility contract:

- Arrow and page navigation can move focus from fixed options to `Other`.
- While `Other` is focused, printable characters edit the custom answer even
  when they match numeric option shortcuts.
- Pressing Enter on an empty focused `Other` row keeps the prompt open instead
  of silently submitting a fixed option.
- Multi-select prompts preserve already selected fixed options when a focused
  `Other` answer is typed and submitted.
- Fixed numeric shortcuts continue to select or toggle listed options when
  `Other` is not focused.

Regression coverage lives in `crates/puffer-tui/src/tests/user_question.rs`.
