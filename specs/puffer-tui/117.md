# Connect Connector Catalog Picker

The TUI slash popup now assists deterministic connector setup after users type
`/connect `.

## UX

- Typing `/connect ` lists built-in connector templates.
- Typing connector terms such as `/connect telegram personal` filters by slug,
  description, skill, action names, auth state, and trigger capability.
- Selecting a connector fills `/connect <connector-slug> <suggested-connection>`
  so the command is deterministic and ready to submit or edit.
- The popup hides after an exact connector slug is followed by a second token,
  preserving free-form custom connection names.

This is a TUI-only discoverability change. It does not change `/connect`
parsing, connector auth, connector runtime behavior, Corbina, workflow storage,
or webhook behavior.
