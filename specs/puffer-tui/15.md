# Honor AskUserQuestion numeric shortcuts

## Purpose

`AskUserQuestion` overlays render selectable options with numeric shortcuts, so
the advertised keys must work during normal keyboard-first use.

## Design

The user-question overlay now routes printable numeric shortcuts through the
same list-selection primitive used by permission prompts. Single-select
questions answer immediately, while multi-select questions toggle the matching
option and still require Enter to submit the final set.

## Logic

- Number keys `1` through `9` select matching visible options.
- Single-select shortcuts confirm the current question.
- Multi-select shortcuts toggle the target option without prematurely closing
  the prompt.
- Existing arrows, Space, Enter, Esc, and Ctrl-C behavior is unchanged.

## Contracts

No tool schema or transcript format changes. This only fixes TUI prompt
keyboard routing so displayed shortcuts match runtime behavior.
