# Pentest Report Module Split

## Summary

Pentest report data structures and parsing helpers now live in
`pentest_report.rs`.

## Contract

`pentest.rs` remains the TUI state machine and persistence owner. The extracted
module owns report structs, report parsing, verified category/root-cause
derivation, and status continuation checks used by the state machine.

## Compatibility

This is a module-boundary change only. Pentest state, report artifacts,
compaction behavior, and command handling remain unchanged.
