# Workflow Action Delete Popup

The TUI slash popup now surfaces existing workflow action bindings for cleanup.

- Typing `/workflows delete `, `/workflows remove `, or `/workflows rm ` can
  search live workflow action bindings and complete to
  `/workflows delete <binding-slug>`.
- Delete popup rows index binding slug, description, connection, connector,
  status, action type, target, and filter pattern.
- The row description shows enough context to distinguish similar append,
  monitor, or connector-action bindings before deleting one.
- The existing `/workflows actions [query]` dashboard output remains the
  text-list view; the popup only adds direct command completion.

This change is TUI-only. It does not alter workflow binding storage, delete
semantics, Corbina behavior, or connector runtime behavior.
