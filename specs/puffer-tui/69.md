# Slash popup workflow connector search

The TUI slash-command popup now indexes command descriptions and argument
hints, not only command names and aliases.

## UX

- Searching `/connector` shows workflow-relevant commands such as `/connect`
  and `/workflows` even if the user does not know the exact command name.
- Searching `/connector-slug` finds `/connect` from its argument hint.
- Name and alias matches still rank above description and argument-hint
  matches.
- The popup still disappears after a command plus a trailing space, so argument
  entry such as `/model ` is not interrupted.

This is TUI command discovery only. It does not change command execution,
connector auth, workflow storage, or webhook behavior.
