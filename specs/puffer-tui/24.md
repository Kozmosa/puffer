# AskUserQuestion option previews

The TUI preserves the optional `preview` field on `AskUserQuestion` options and
renders the preview for the currently highlighted single-select option.

Compatibility contract:

- `preview` remains optional and empty previews are ignored.
- Preview text is rendered through the existing markdown renderer.
- Previews are shown for single-select questions only; multi-select questions
  continue to render as checkbox-style option lists.
- The preview pane is capped so a large preview cannot consume the whole
  composer area.
- Moving the selection updates the rendered preview without changing the
  answer payload.

Regression coverage lives in `crates/puffer-tui/src/tests/user_question.rs`.
