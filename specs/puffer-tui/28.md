# AskUserQuestion custom answers

The TUI supports the `AskUserQuestion` custom-answer path that the tool
contract exposes as "Other".

Compatibility contract:

- Fixed numeric option shortcuts continue to select or toggle listed options.
- Typing non-shortcut text while a question prompt is open records a custom
  answer on the prompt overlay instead of mutating the composer draft.
- Single-select questions submit the custom answer when custom text is present.
- Multi-select questions append custom text to the selected option labels.
- The inline prompt displays the custom-answer buffer while preserving the
  user's composer draft after the question is answered.

Regression coverage lives in `crates/puffer-tui/src/tests/user_question.rs`.
