# Live Connection Slash Completions

The TUI slash popup now prefers configured connector connections when building
workflow and connector command suggestions.

- `/workflows new ...` and `/workflows append ...` can surface live connection
  slugs from the installed subscription manager before falling back to planned
  connector defaults.
- `/monitor ...` can surface live connection slugs with the exact command the
  monitor flow expects.
- `/connect ...` can surface live connection repair commands such as
  `/connect <connector> <connection>`.
- Planned connector catalog rows remain available when no configured
  connection matches the query.
