# Usage Overlay Month-End Reset

The Claude subscription usage overlay now computes extra-usage reset dates from
the first day of the current month before advancing to the next month. This
prevents month-end dates such as May 31 from attempting to construct invalid
calendar dates such as June 31 and hiding the extra-usage section.

The inline usage overlay tests were moved into `src/usage/tests.rs` so the
production module stays below the repository line limit. The new regression
test covers the May 31 to June 1 transition directly, while the existing render
test verifies that the extra-usage panel remains visible in the TUI.
