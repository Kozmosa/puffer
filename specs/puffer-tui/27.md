# Discovery-only provider pickers

The TUI provider pickers include providers that can discover models even when
they do not ship bundled model entries.

Compatibility contract:

- Providers with bundled model entries stay visible.
- Providers with `discovery` metadata and no bundled models stay visible in
  onboarding provider selection.
- `/login` also includes authenticated providers with discovery metadata and no
  bundled models.
- Providers with neither bundled models nor discovery metadata remain hidden
  from model-oriented pickers.
- Selecting a discovery-only provider still uses the existing best-effort model
  refresh path before opening model selection.

Regression coverage lives in `crates/puffer-tui/src/onboarding/mod.rs`.
