# Initial model selection

The TUI treats a selected provider without a selected model as incomplete setup.

Compatibility contract:

- A session with no provider still opens provider setup.
- A session with a provider but no model still opens model setup.
- A provider/model selector must include the current provider id and a model id
  known to that provider before setup is considered complete.
- Provider model discovery and missing-auth checks keep their existing order.

Regression coverage lives in `crates/puffer-tui/src/onboarding/mod.rs`.
