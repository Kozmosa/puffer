# Linear Webhook Connector Preset

## Summary

The built-in connector catalog includes `linear-webhook`, a Linear issue and
project webhook preset backed by the existing `puffer serve` webhook transport.

## Design

- The connector slug and suggested connection slug are both `linear-webhook`.
- Runtime hints include `serve`; the connector has no subscriber manifest,
  command runtime, workflow trigger support, or connector actions.
- The output schema describes normalized Linear webhook payloads with type,
  action, actor, message, and optional URL fields.
- Shared schema helpers moved into `catalog/schemas.rs` so the catalog module
  stays under the Rust source size limit while preserving existing action and
  message schemas.

## Compatibility

This is additive to the catalog. The existing generic `webhook` and
`github-webhook` presets keep their slugs, schemas, setup commands, and
serve-mode behavior.
