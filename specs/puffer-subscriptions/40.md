# New Relic Connector Template

The built-in connector catalog now includes `newrelic-webhook`, a setup-only
serve preset for New Relic issue and alert webhook notifications.

## Template

- Slug: `newrelic-webhook`.
- Suggested connection slug: `newrelic-webhook`.
- Skill: `newrelic-webhook`.
- Binary: `puffer connector webhook`.
- Runtime hints: served by `puffer serve`; no connector protocol command.
- Capabilities: no auth requirement, no subscription trigger, no proxy, and no
  outbound actions.
- Output schema kind: `newrelic_issue`.

## Compatibility

The template participates in connector search and deterministic setup without
changing workflow runtime semantics. It remains setup-only until the webhook
connector is promoted to a first-class workflow trigger.
