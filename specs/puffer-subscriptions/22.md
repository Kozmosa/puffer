# Connector Connection Hints

## Summary

Connector templates now expose a shared deterministic connection-slug hint so
workflow UX surfaces can show the exact `/connect <connector> <connection>`
shape before a connection exists.

## Behavior

- `suggested_connection_slug(connector_slug)` returns stable defaults for the
  built-in connectors.
- `telegram-login` maps to the established personal-account default
  `telegram-user`.
- Slack, Lark, email, bot, and custom connector slugs default to their connector
  slug when no more specific connection name is known.

## Compatibility

This helper does not change connector records, authentication, subscription
startup, or action execution. It is only a display and draft-planning contract
for clients that need deterministic `/connect` guidance.
