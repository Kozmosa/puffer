# Shopify Webhook Connector Preset

## Summary

Adds `shopify-webhook` to the built-in connector catalog as a serve-backed
setup-only webhook preset. The connector is intended for Shopify order,
product, customer, inventory, fulfillment, and app lifecycle JSON webhook
deliveries.

## Catalog Contract

- `connector_slug`: `shopify-webhook`
- `suggested_connection_slug`: `shopify-webhook`
- `skill`: `shopify-webhook`
- `binary`: `puffer connector webhook`
- `requires_auth`: `false`
- `can_subscribe`: `false`
- `can_proxy_agent`: `false`
- `actions`: none
- `output_schema`: `shopify_event_output_schema`

The preset does not claim connector protocol runtime support. It is surfaced so
`/workflows connectors`, `/connect`, and GUI connector pickers can guide users
to configure the generic serve-mode webhook connector with Shopify-specific
defaults.

## Compatibility

The Shopify preset follows the existing setup-only webhook preset model used by
Asana, GitHub, GitLab, Jira, Linear, Stripe, and Trello. It remains setup-only
until serve-mode webhooks are first-class workflow triggers.
