# Figma Webhook Catalog Metadata

## Scope

Adds `figma-webhook` to the built-in workflow connector template catalog.

## Design

- The connector is a serve-mode webhook preset with no subscriber command, no
  workflow trigger runtime, no proxy-agent support, and no outbound actions.
- The output schema describes normalized Figma event fields including event,
  file key, file name, actor, comment id, version id, node id, status, message,
  and URL.
- Suggested connection slug remains deterministic: `figma-webhook`.

## Compatibility

The connector is setup-only and uses the same generic webhook serve contract as
the existing webhook presets.
