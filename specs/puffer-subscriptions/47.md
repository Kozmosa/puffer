# Built-In Connector Catalog Without HTTP Ingress

## Change

The subscription catalog removes the built-in HTTP ingress connector templates.

## Contract

- Built-in templates now cover Telegram, Discord, Lark, Matrix, Slack, and
  email only.
- Ingress-specific output schemas are removed from the catalog schema module.
- Suggested connection slugs remain deterministic for the surviving built-ins.
- Serve runtime hints apply only to Telegram, Discord, and Matrix bot
  connectors.

## Compatibility

Workflow trigger support for subscriber-backed connectors is unchanged. Existing
connection records with deleted connector slugs are not migrated by this catalog
change; they simply no longer resolve to a built-in template.
