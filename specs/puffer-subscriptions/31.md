# Stripe Webhook Connector Preset

The built-in connector catalog includes `stripe-webhook`, a setup-only
serve-mode preset for Stripe payment and billing webhooks.

## Contract

- The connector uses the shared `puffer connector webhook` binary.
- It is not a subscriber, agent proxy, or connector protocol action source.
- Runtime hints identify it as `serve`.
- Suggested connection slug: `stripe-webhook`.
- Skill: `stripe-webhook`.
- Output schema identifies `stripe_event` payloads with event type, object type,
  object id, account, message, and URL fields.

## Compatibility

The preset is additive. Existing webhook presets and generic `webhook`
configuration keep their slugs, schemas, setup commands, and runtime behavior.
