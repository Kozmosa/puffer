# puffer-media 05: Typed Relaydance video parameters

Resolved media requests now carry each provider request field's `wire_type`
alongside the string selection value.

The `relaydance_video` adapter uses that type metadata when building JSON:

- string fields remain JSON strings;
- number fields are parsed and sent as JSON numbers;
- invalid or non-finite numeric values fail before the HTTP request.

This keeps WorldRouter Seedance `duration` aligned with the documented task
body while preserving string encoding for resolution and metadata fields.
