# Proto Optional Build Flag

## Purpose

`puffer-cli` verification requires `puffer-runner-grpc` to build on protoc
versions that still gate proto3 optional fields.

## Design

The build script passes `--experimental_allow_proto3_optional` through
`tonic_build::configure().protoc_arg(...)` before compiling
`proto/tool_runner.proto`.

## Compatibility

This is build-only. The proto schema, generated service shape, wire format, and
runtime behavior are unchanged.
