# Env Var Test Module Split

## Summary

The environment-variable registry tests now live in `env_vars_tests.rs`.

## Contract

`env_vars.rs` remains the source of the registry and public lookup APIs. The
test module still verifies unique names and non-empty descriptions, but it is
compiled from a sibling file so the registry source stays below the repository
line cap.

## Compatibility

This is a structural test split only. Environment variable metadata and lookup
behavior are unchanged.
