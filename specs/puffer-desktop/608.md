# Task History Popup

- The Tasks screen adds a History button next to Configure.
- History opens a modal that loads `task_monitor_history_list`, showing received monitor messages in a left sidebar and details on the right.
- The right pane shows triage outcomes from the selected message action log and ignore-analysis records from ignored monitor tasks linked by `monitor_envelope_id`.
- Agent rows show token spend when usage is available; missing usage is shown as unavailable rather than inferred.
- The popup includes refresh and close controls, loading and empty states, compact source/timestamp labels, and raw message payload text for inspection.
- Desktop tests cover the popup rendering received messages, triage outcomes, ignore-filter outcomes, and linked ignore-analysis text.
