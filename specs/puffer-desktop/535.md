# Connector Row Append Actions

Corbina's Pipelines connector catalog now lets trigger-capable connector rows
create file-append workflow bindings directly from search results.

- Connector rows expose an append action beside setup and draft actions when the
  connector can start workflow triggers.
- The row action uses the same deterministic connection name as the connector
  row, including any staged custom name inferred by connector search.
- Append path and filter pattern reuse the connector search query inference, so
  searches such as `append email hi /tmp/email.log` create the matching
  `/workflows append` binding without selecting the connector detail first.
- Setup-only connectors keep the append action disabled.

This change is UI-only. It does not change connector runtime behavior,
workflow binding storage, TUI behavior, or connector auth semantics.
