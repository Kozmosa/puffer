# 358. Sidebar Account Entry

The sidebar account chip must not look like an action while staying inert.

- The signed-in user chip is now a native button with a unique accessible name.
- Activating it opens Settings so account/provider status and sign-out controls
  are reachable from the sidebar, without polluting the primary Settings
  navigation accessible name.
- The trailing More icon remains a navigation affordance, not a dead decorative
  control.

Playwright coverage clicks the account chip and verifies the Settings page opens
with the account controls visible.
