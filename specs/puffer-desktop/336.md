# 336. Deployment Public URL Open Action

Deployment detail controls must not render inert action buttons.

Contracts:

- The detail-header `Open` button opens the selected deployment's public URL in
  a new browser tab or window.
- Hostnames without a scheme are normalized to `https://` before opening.
- Deployments without a public URL keep the user in place and report that no
  URL is available instead of silently doing nothing.
- The status message uses the same live detail action region as redeploy
  feedback so pointer and keyboard users receive immediate confirmation.

Playwright coverage stubs `window.open`, clicks the deployment detail `Open`
button, verifies the normalized URL and target features, then selects an
internal infrastructure deployment and verifies the unavailable-URL feedback
without opening an extra tab.
