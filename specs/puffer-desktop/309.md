# 309. PDF Preview Zoom Contrast

## Summary

The desktop Files PDF preview keeps zoom controls visible and makes page-limit
status messages visually distinct from the preview background.

## Behavior

- PDF zoom controls stay sticky at the top of the scrollable preview.
- Rendered PDF canvases are no longer capped to the preview width, so zooming in
  increases the actual page size and the shell can scroll horizontally.
- Long-document status pills use an accent-tinted background and foreground
  contrast instead of blending into the preview shell.

## Compatibility

The renderer still limits previews to the first 20 pages, preserves the text
fallback, and keeps the existing 50% to 200% zoom range.
