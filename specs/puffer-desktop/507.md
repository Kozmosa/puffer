# Shopify Connector Catalog GUI

## Summary

Adds the Shopify webhook preset to the Corbina/Pipelines connector catalog test
fixture and search coverage.

## UI Contract

The catalog row exposes:

- slug: `shopify-webhook`
- description: `Shopify order, product, customer, and inventory webhook preset backed by puffer serve`
- skill: `shopify-webhook`
- runtime hint: `serve`
- trigger state: setup-only / no trigger
- command: `/connect shopify-webhook shopify-webhook`

Search terms such as `shopify order product webhook`, `serve webhook`, `serve`,
`no trigger`, and `setup-only webhook` include the Shopify row. Selecting it
shows the same details and deterministic connect command as the other
serve-backed webhook presets.

## Compatibility

Connector catalog result totals increase from 18 to 19. Serve-backed connector
counts increase from 11 to 12, and setup-only webhook counts increase from 8 to
9. Existing connection and workflow draft UX is unchanged.
