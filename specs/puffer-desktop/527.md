# Corbina searchable question term matching

Searchable `AskUserQuestion` prompts in Corbina now match multiple query terms
against the combined option label and description.

## UX

- A connector picker question can be filtered with natural phrases such as
  `personal connector` instead of requiring one contiguous substring.
- Terms may match across the option label and description, matching the TUI
  searchable-question behavior.
- Existing answer selection and selected-answer clearing still use the same
  matching predicate, so stale selections are removed when a new query no longer
  matches them.

This only changes Corbina pending-question filtering. It does not change
connector auth, connector manifests, workflow storage, or webhook behavior.
