# Puffer Desktop Update 630

## Summary

CEF release builds now repair a partially-applied ChromeMainDelegate resource
bundle hook by adding the required header include independently.

## Design

- `scripts/release.sh` inserts `ui/base/resource/resource_bundle.h` whenever the
  ChromeMainDelegate header lacks it.
- The include repair runs separately from the hook declaration insertion, so a
  checkout where CEF partially applied only the declaration is still fixed.

## Compatibility

This keeps CEF's resource bundle delegate hook compiling against Chromium
checkouts where the upstream CEF patch succeeded only partially.
