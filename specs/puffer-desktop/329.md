# 329. Ask Puffer Composer Submit

## Summary

The Deployments Ask Puffer composer is an active interaction instead of a dead
text box with a decorative send button.

## Behavior

- Typing a prompt and pressing Send appends a user message and a scoped Puffer
  response for the selected deployment.
- Pressing Enter in the composer performs the same submit action.
- Switching deployment environments clears the local draft and local Ask Puffer
  turns so prompts do not leak between environments.

## Verification

- Playwright opens Deployments, sends prompts with both the Send button and
  Enter, verifies the prompt appears once as a user message, checks the generated
  response references the selected deployment, and confirms the composer clears
  after each submit.
