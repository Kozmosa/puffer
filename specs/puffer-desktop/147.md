# Connect Project Picker Mode Isolation

## Scope

The Connect Project modal has separate Local and Remote modes. The local
directory picker belongs only to Local mode and must not remain visible after
the user switches to Remote, where the fields refer to a different filesystem.

## Change

- Mode switches now route through a helper that closes the directory picker
  before changing modes.
- Closing the picker increments the picker load generation and clears loading
  and error state so late `list_dir` responses cannot refresh hidden local
  picker state.
- Escape, close, and "Use this directory" now use the same close helper.
- A Playwright regression test opens the local picker, switches to Remote, and
  verifies the picker is gone while the SSH target field is visible.

## Compatibility

Local browsing behavior is unchanged while the modal remains in Local mode.
Remote project creation no longer shows stale local picker UI.
