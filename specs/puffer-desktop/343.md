# 343. Pipeline Wiring Connected State

Pipeline wiring controls must not leave already connected output targets looking
actionable.

Contracts:

- The `Send selected output to` list disables a target when that target already
  depends on the selected agent node.
- Disabled connected targets expose `aria-pressed="true"` and a visible
  `connected` affordance.
- Unconnected targets remain clickable, and clicking one immediately transitions
  it into the disabled connected state.

Playwright coverage opens Pipelines, checks that the existing Codex to Claude
wire is disabled as connected, clicks the Codex to Puffer output target, and
verifies the target becomes disabled and pressed.
