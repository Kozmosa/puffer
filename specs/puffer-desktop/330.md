# 330. Deployment Search Interaction

## Summary

The Deployments screen Search control opens a working filter instead of a dead
toolbar button.

## Behavior

- Clicking Search opens and focuses a deployment search box.
- Typing filters environments by name, provider, branch, status, region, URL,
  alert text, workspace name, and workspace role.
- When a filter hides the selected deployment, the detail pane follows the first
  visible match.
- Empty results show an inline status message.
- Escape closes search and restores the full environment list.

## Verification

- Playwright opens Deployments, activates Search, verifies focus, filters to a
  Cloudflare environment, checks the detail pane follows the filtered match,
  verifies the empty state, and confirms Escape restores all environment rows.
