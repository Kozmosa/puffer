# 590 - Network Proxy Editor Layout

Updates the Settings -> Network proxy controls to match the compact connection
editor layout.

- The Proxy enable switch is aligned to the right side of the settings row.
- The Proxy list Add proxy action lives in the section header and aligns right.
- Add proxy starts with a `socks5://127.0.0.1:7890` draft without showing an
  endpoint preview line.
- Edit proxy and Add proxy submit with the standard default button style.
- Host and Port share one two-column row; Username and Password share another.
- Port uses a text input with numeric input hints instead of a browser number
  spinner control.
