# Stable Provider Picker Hover

## Summary

Provider selection controls now use stable hover feedback that does not resize
or visually pump the option under the cursor.

## Design

Shared modal chrome no longer animates through `scale()` and the scrim avoids
backdrop blur so opening provider dialogs does not trigger expensive filtered
repaints. Provider segmented controls retain their existing fixed geometry and
base font weight on hover/active states, relying on background and border
changes for feedback instead.

`NewSessionModal` provider choices now declare explicit color-only transitions
so hover remains smooth and predictable.

## Contracts

- Provider option hover does not change button width, height, or font weight.
- Active provider state remains visible through background/border treatment.
- Modal entry animation still fades and slides without scaling its children.
- The change applies to new-agent and create-project provider selection, plus
  pipeline provider cards that used the same hover weight jump.
