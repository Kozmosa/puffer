# 282. Workspace Agent Card Full Titles

## Summary

Workspace project agent cards now keep the complete session title in the DOM and
accessible button metadata. Visual truncation is handled only by the existing
responsive CSS ellipsis.

## Behavior

- Long session titles are no longer shortened in component state.
- The card tooltip continues to include the full title, status, and elapsed
  time.
- Agent cards use explicit `type="button"` semantics.
- The existing `.pf-pw-agent .title` overflow rules remain responsible for
  drawing ellipses when the available width is too small.

## Compatibility

This does not change card ordering, project grouping, status chips, or click
routing. It only removes data-level title truncation so wider layouts, search,
copy, and assistive technologies can access the full title.
