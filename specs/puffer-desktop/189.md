# 189. Modal-Safe Shell Shortcuts

## Summary

Shell-level keyboard shortcuts now stay inactive while modal dialogs are open.

## Design

- The global Settings shortcut (`Cmd/Ctrl + ,`) still opens Settings from the
  normal workspace shell.
- When a modal such as New Agent or Switch Workspace is open, the shortcut is
  ignored so focus-trapped modal flows cannot accidentally navigate the
  background screen.
- The guard also checks for any active ARIA modal dialog, keeping future modal
  components covered without per-dialog wiring.

## Tests

Playwright coverage asserts that `Cmd/Ctrl + ,` opens Settings from the normal
workspace and remains inert while the New Agent modal is visible.
