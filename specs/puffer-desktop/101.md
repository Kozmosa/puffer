# Connect Project Picker Escape

## Scope

The Connect Project modal contains a nested directory picker. Escape should
close the topmost picker first instead of discarding the whole parent modal.

## Change

- `ConnectProjectModal.svelte` checks `pickerOpen` in the global Escape
  handler.
- When the picker is open, Escape closes only the picker and preserves the
  parent modal fields.
- While the picker is loading, Escape is consumed so the parent modal does not
  disappear underneath a pending directory request.

## Compatibility

The modal close button and parent Escape behavior are unchanged when no nested
picker is open.
