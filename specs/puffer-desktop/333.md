# 333. Deployment Draft Creation Modal

The Deployments top bar must not show a dead `New deployment` action. It opens
a focused modal that creates a local draft deployment in the mock desktop
surface.

## Contract

- `New deployment` opens a modal dialog and moves focus into the service-name
  field.
- Escape closes the modal without changing the deployment list.
- `Create deployment` stays disabled until a service name is present.
- Creating a draft closes the modal, restores focus to the opener, inserts the
  draft at the top of the deployment list, updates the environment count, and
  selects the draft on the Deploys tab.
- Drafts stay local until the daemon exposes provider deployment creation RPCs.

## Coverage

Playwright opens Deployments, verifies Escape cancellation leaves the list at
six rows, fills the draft form, creates a Fly preview deployment, and asserts
the new row, selected detail header, Deploys tab, and top-bar count all update.
