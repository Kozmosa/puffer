# Browser DevTools Empty Tab Reset

## Summary

When the browser daemon reports an empty tab list, the Browser pane now closes
the DevTools side panel and resets its internal view to Console.

## Behavior

- Empty tab-list events still clear all tabs, status, pending commands, pointer
  state, canvas contents, and persisted browser tabs.
- DevTools is no longer left open beside the `No pages` state.
- Opening a new browser tab after the empty state starts from the default
  Console DevTools view if the user opens DevTools again.

## Tests

`desktop-ui.spec.ts` extends the stale-tab clearing regression to open DevTools
before the daemon clears all tabs and asserts that the DevTools panel is removed
with the tabs.
