# 311. Closing Sessions Clears Restore Target

## Summary

Closing an agent detail view now clears the desktop remembered-session restore
target while keeping the session in workspace history.

## Behavior

- Selecting a session still records it when `Remember last session` is enabled.
- Pressing `Close session` returns to the workspace and removes the
  `puffer-desktop:remembered-session` entry.
- Reloading the desktop shell after closing stays on the workspace instead of
  reopening the closed detail view.
- The closed session remains available in the history list and can be reopened
  manually.

## Compatibility

This only changes the restore target. It does not delete session metadata,
transcripts, active-agent rows, or workspace history entries.
