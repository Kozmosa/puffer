# Contacts Avatar Metadata Rendering

## Design

The contacts screen renders saved avatar values only as image sources. Compact
rows and selected-contact metadata show a short avatar status label instead of
printing the raw stored avatar string.

## Compatibility

Saved contact payloads and daemon contracts are unchanged. Existing URL and
data URI avatars still render through the existing image element, but large
base64 data URIs are no longer exposed as visible text in the contacts UI.
