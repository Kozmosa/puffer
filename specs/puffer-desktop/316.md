# Review Agent Detail State

## Contract

Agent detail surfaces must preserve the backend review status through every
visible state affordance. A session with `activityStatus: "review"` renders the
status pill as `review` and keeps the header Puffer orb in
`data-state="review"` instead of falling back to idle.

## UI Behavior

- `agentPufferState` maps review agents to the `review` Puffer state.
- The review Puffer state uses a slow, lightweight breathe animation so it reads
  as ready-for-review without implying that a turn is still running.
- Existing busy overrides still win: pending permissions become `awaiting`, and
  live turns become `thinking` or `running`.

## Verification

Playwright coverage opens a daemon-backed review session and asserts both the
review status pill and the detail Puffer orb state.
