# Puffer Desktop Update 634

## Summary

Linux CEF release builds now restore CEF's friend access to Chromium tab helper
attachment.

## Design

- `scripts/release.sh` adds an idempotent TabHelpers compatibility patcher.
- The patcher includes CEF build flags in `tab_helpers.h`, forward declares
  `CefBrowserPlatformDelegateAlloy`, and friends it when CEF is enabled.
- The repair keeps `TabHelpers::AttachTabHelpers` private to Chromium and CEF's
  known caller instead of making the helper public.

## Compatibility

Chromium's normal desktop tab-helper access model remains private and
friend-based. CEF's Alloy browser delegate can still adopt WebContents with the
full Chrome tab helper set during embedded browser creation.
