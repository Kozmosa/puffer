# Browser Canvas Find Shortcut Isolation

The agent-detail global find shortcut must not open while keyboard focus is
inside the embedded Browser canvas. Browser canvas key events belong to the
managed browser session, including `Ctrl+F` / `Cmd+F`, and should continue
to be handled by the focused browser surface instead of being consumed by the
surrounding agent view.

The global shortcut target filter treats `.pf-browser-canvas` like other
interactive editing surfaces. Regression coverage verifies that a focused
canvas keeps the app find closed and preserves canvas focus.
