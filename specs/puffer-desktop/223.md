# Deployments Navigation Entry

The desktop shell exposes every routed primary screen through the sidebar.
`deployments` is already part of `ScreenId` and is rendered by `App.svelte`, so
the sidebar must include a Deployments item instead of leaving the screen
reachable only through persisted or programmatic state.

The navigation entry uses the existing rocket icon and the same accessible
button label behavior as the other primary destinations. Regression coverage
checks both normal sidebar access and the 720px desktop minimum-width sidebar.
