# 341. Deployment Add Provider Draft

Deployment provider controls must not expose an enabled Add action that drops
clicks.

Contracts:

- The Providers pane `Add provider` button opens a focused local form.
- The form requires a provider name and connection note before submission.
- Added providers are inserted at the top of the selected deployment's
  integration list with selected type and connection status.
- Duplicate provider names in the selected deployment stay disabled locally.
- The pane live status reports the added provider and selected deployment.

Playwright coverage opens the Providers tab, creates a degraded webhook
provider, checks that the local card count increases, and verifies the rendered
name, note, status, and live status text.
