# Linear Connector Catalog Details

## Summary

The Pipelines connector picker surfaces the `linear-webhook` preset and shows
selected connector setup metadata in Corbina.

## Behavior

- The fake daemon fixture includes `linear-webhook`, so GUI tests cover the
  production connector count and search behavior.
- Search terms such as `linear issue webhook`, `serve`, `setup-only webhook`,
  and `no trigger` include the Linear preset.
- Selecting a connector now shows a compact detail row with slug, description,
  auth/trigger status, skill, runtime hints, and visible action chips before
  the command controls.
- The selected Linear setup displays `/connect linear-webhook linear-webhook`
  and remains setup-only with `serve` and `no trigger` tags.

## Compatibility

The daemon snapshot contract is unchanged. Connectors without runtime hints or
actions still render the existing command controls and simply omit those chips.
