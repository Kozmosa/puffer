# Desktop side-panel tab safety

The Agent Detail side panel is reserved for passive companion views that do
not mount a second copy of backend-connected panes.

Compatibility contract:

- Command-clicking the Diff tab still opens it beside the current main tab.
- Command-clicking the active tab is a no-op.
- Browser and Terminal tabs are not duplicated into the side panel, avoiding
  competing subscriptions, resize calls, and PTY/browser focus changes for the
  same session resources.
- Side-panel rendering continues to reuse the same Agent Detail content path
  for allowed passive tabs.

Regression coverage lives in `agent-detail-ui.spec.ts`.
