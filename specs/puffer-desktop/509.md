# Pipelines PagerDuty Connector Catalog Coverage

## Change

The Corbina/Pipelines fake daemon and catalog tests now include the
`pagerduty-webhook` setup-only connector row.

## UX Contract

- Global connector counts increase from 20 to 21.
- Searching `pagerduty incident webhook` surfaces only the PagerDuty setup row.
- The row advertises `serve` and `no trigger`.
- Selecting the row shows `skill:pagerduty-webhook` and the deterministic setup
  command `/connect pagerduty-webhook pagerduty-webhook`.
- Existing filter presets continue to distinguish trigger-ready, setup-only,
  action-capable, and serve-backed connector rows.

## Compatibility

No production desktop state shape changes are required. The fake daemon remains
an executable mirror of the daemon connector catalog DTO used by the UI tests.
