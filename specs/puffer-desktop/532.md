# Corbina Connection Append Actions

Corbina's pipeline connector picker exposes direct append-workflow creation from
existing connection rows.

- Each trigger-capable connection row now includes a file action that creates an
  enabled `file_append` workflow binding through the existing
  `workflow_binding_create` daemon API.
- The row action uses the deterministic command shape already indexed by
  connector search: `/workflows append <connection> /tmp/<connection>.log`.
- Creating the binding selects the connection's connector in the detail panel,
  updates the append path preview, and refreshes the workflow snapshot returned
  by the daemon.
- Unsupported connections keep the append action disabled, matching draft
  trigger behavior.
