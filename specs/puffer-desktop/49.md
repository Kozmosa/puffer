# Filter workspace search results

The workspace search box now filters the visible project rows and agent cards
instead of acting as a decorative input. Search matches project names, paths,
branches, remotes, agent names, titles, branch labels, status text, and model
labels.

Compatibility contract:

- Empty search keeps the existing workspace list unchanged.
- A project-level match shows the project with all of its visible agents.
- An agent-level match shows only matching agents inside that project row.
- No-match search renders an empty state with a clear-search action.
- The search input is labeled for automation and screen readers.

Regression coverage lives in `workspace-ui.spec.ts` with two daemon-backed
projects and a no-results clear-search path.
