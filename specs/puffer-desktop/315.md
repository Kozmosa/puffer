# 315. PDF Preview Zoom Stays Live

## Summary

PDF zoom controls now resize already rendered pages immediately instead of
clearing the document and re-rendering every page on each click.

## Behavior

- Zoom in, zoom out, and reset controls remain sticky at the visible top-left
  of the PDF preview.
- Zooming a long PDF preserves rendered pages and the page-limit status while
  the visible canvas sizes update.
- The page-limit status uses a high-contrast pill so messages such as
  "Showing first 20 of 29 pages." do not blend into the preview background.

## Compatibility

PDF.js still renders the source document at the existing base render scale.
The change only affects how rendered canvases are displayed after zoom changes.
