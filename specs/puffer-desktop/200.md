# Active Agent Project Disambiguation

## Summary

Active agents now keep sessions from different workspace paths separated even
when those paths share the same folder basename.

## Design

`App.svelte` keeps the short project label for unique basenames. When multiple
folder groups share the same label, the active-agent sidebar uses the full
workspace path as the group label so `Sidebar.svelte` groups them separately.

## Contracts

- Sessions from different folder paths must not collapse into one Active agents
  group just because their final path segment matches.
- Unique project labels stay compact.
- Collapsing one duplicate-named project group must not hide sessions from the
  other path.
