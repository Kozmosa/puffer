# 443. Auth-Free Provider Readiness In Settings

## Summary

Settings marks auth-free local providers as ready instead of disconnected.

## Design

- Provider readiness now includes agent-capable providers with explicit empty `authModes`.
- The connection summary shows ready agent providers when local auth-free providers are available.
- Auth-free provider cards show `Ready`, a `local` pill, and `No credentials required`.

## Compatibility

Credential-backed providers keep their existing connected/disconnected copy and actions. Providers that require credentials still show connect controls until authenticated.

## Tests

Playwright covers both the existing connected/disconnected provider cards and an auth-free Ollama provider card.
