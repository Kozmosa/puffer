# 340. Deployment Add Memory Note Draft

Deployment memory controls must not expose an enabled Add action that drops
clicks.

Contracts:

- The Memory pane `Add note` button opens a focused local form.
- The form requires a title and body before submission.
- Added notes are inserted at the top of the selected deployment's memory list
  with selected kind, confidence, tags, and a local manual source.
- The note count and kind filter counts update immediately.
- The pane live status reports the added note and selected deployment.

Playwright coverage opens the Memory tab, creates a runbook note, checks the
updated note count, live status, rendered note body/tags/source, and verifies
the Runbook filter includes the new local note.
