# Contact Inference Trace Default

## Design

The Contacts inference trace starts collapsed for each new run. Users can expand
the trace to inspect prompt and tool activity, but proposal review remains the
default focus when the modal opens or reruns.

## Architecture

`Contacts.svelte` keeps the existing shared `MessageBody` and `ToolCard`
renderers. Only the local `inferTraceCollapsed` state default changes; trace
subscription, event adaptation, proposal rendering, and close behavior are
unchanged.

## Compatibility

The `contacts_infer` RPC and trace event contract are unchanged. Existing trace
events still render after expansion.
