# Pipelines unique connector action detail

Corbina's Pipelines connector catalog now mirrors the terminal workflow
dashboard detail behavior for unique connector searches.

## UX

- General connector catalog rows stay compact and still show at most three
  action chips plus a hidden count.
- When a non-empty connector search narrows the catalog to one connector, that
  row expands to show every action slug in the row itself.
- Users can inspect the full action surface from search results before opening
  the selected connector setup panel.
- Connector row rendering reuses the existing `connectionsByConnector` map
  instead of filtering all connections for every connector row.

This is a Pipelines UI rendering and search-responsiveness change only. It does
not add connector templates, connector auth behavior, workflow storage, or
webhook behavior.
