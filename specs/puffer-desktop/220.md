# Model Picker Escape Close

The agent model picker must behave like a small popup, not a persistent panel.
When the menu is open, pressing Escape closes it, prevents the key from leaking
to lower-priority app shortcuts, and returns focus to the trigger button so the
same keyboard/mouse path can reopen it immediately.

This keeps the picker consistent with desktop modal and picker behavior while
leaving outside-click close behavior unchanged. Regression coverage lives in the
chat-session Playwright suite.
