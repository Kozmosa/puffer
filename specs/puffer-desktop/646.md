# Secrets Settings Tab

The desktop settings screen includes a `Secrets` tab for managing encrypted
agent secrets.

- The tab displays vault path/key-source metadata and a list of saved secret
  summaries without rendering raw values after save.
- Manual creation captures label, optional username/origin, and a password input
  value, then calls `save_secret`.
- Chrome import calls `import_chrome_secrets` and reports non-secret import
  counts.
- Delete calls `delete_secret` for a single summary id.
- The fake daemon and Playwright settings regression cover save/import behavior
  and assert that the raw value is not rendered into visible text.
