# Contacts Screen List Detail Redesign

The desktop Contacts screen now follows the Tasks screen information pattern.

- Saved contacts render as a searchable list/detail workspace with task-style
  top actions, summary metrics, rows, and inspector panel.
- Create and edit actions open a modal contact editor instead of using an
  inline form panel.
- Contact inference opens a modal proposal flow; accepted proposals are promoted
  into the contact editor.
- Contact modals can be dismissed through the close button, Escape, or backdrop
  click even while inference is running.
- Connector candidates are not displayed on the Contacts screen.
