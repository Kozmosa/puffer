# 356. Ask Puffer Quick Actions

Ask Puffer controls must not expose dead quick-action buttons.

- The seeded diagnostic actions now append visible local turns for opening a fix
  PR and rolling back the suspect deploy.
- Composer context chips toggle selected context with `aria-pressed` instead of
  silently doing nothing.
- Submitted Ask Puffer prompts mention the selected context chips and clear the
  chip selection after sending.

Playwright coverage clicks both diagnostic action buttons, toggles logs and
metrics context chips, submits a prompt, and verifies the thread reflects the
selected contexts while the chips reset.
