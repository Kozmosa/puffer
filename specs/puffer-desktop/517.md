# Figma Connector Catalog Row

## Scope

Adds `figma-webhook` to Corbina's Pipelines connector catalog fixture and
search coverage.

## Design

- The fake daemon exposes `figma-webhook` with `serve` runtime hints, no
  trigger support, no actions, and `/connect figma-webhook figma-webhook`.
- Pipelines UI tests verify the connector appears in all connector coverage,
  serve-filter, setup-only webhook-filter, and selected-connector detail flows.
- Result count expectations increase from 28 to 29 total connectors and from
  18 to 19 setup-only webhook results.

## Compatibility

This is test fixture coverage for the desktop GUI. Runtime behavior still comes
from the daemon connector catalog returned by Puffer.
