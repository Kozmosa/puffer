# Opsgenie Pipeline Connector Catalog

Corbina's Pipelines connector catalog now includes the `opsgenie-webhook`
setup-only row in fake daemon fixtures and Playwright coverage.

## Behavior

- The all-connectors count is `26/26`.
- Serve and setup-only webhook filters include `opsgenie-webhook`.
- Searching `opsgenie alert action webhook` surfaces the Opsgenie setup row.
- Selecting the row shows `skill:opsgenie-webhook`, the serve runtime hint, and
  the deterministic command `/connect opsgenie-webhook opsgenie-webhook`.

## Compatibility

Opsgenie remains a setup-only connector in the UI: the row is selectable for
connection setup, but it does not present itself as workflow-trigger ready.
