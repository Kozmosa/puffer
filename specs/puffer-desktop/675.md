# Provider Settings Layout Refresh

- The Providers settings pane now mirrors the Paper-selected structure with a page title, local-model setup card, default-routing controls, and grouped provider auth sections.
- Color treatment stays on existing app tokens and provider visual accents; the change only adjusts layout, copy, spacing, and type scale.
- Default routing now uses a full-width section with side-by-side provider/model selectors and an aligned save action on desktop, collapsing to a single column on small screens.
- Provider auth cards are grouped under Connected providers and Popular providers, with compact two-column desktop cards and one-column responsive fallback.
- The MiniCPM5 local-model card now participates in the section gap rhythm instead of carrying its own bottom margin.
