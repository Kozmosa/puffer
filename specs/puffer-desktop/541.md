# Verified Skills Folder Toggle UX

The Verified Skills Settings pane now treats the Folders switch as a real
remove action. Turning a folder off calls the daemon remove RPC, disables the
row while the request is in flight, refreshes the recognized-skill snapshot,
and restores the current daemon state if removal fails.

When a picked folder is already covered by an existing broader Verified Skills
folder, Desktop shows a local "already covered" note and does not send a
duplicate save request.

Validation:
- `npm run check`
