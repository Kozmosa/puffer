# Pipeline Connection State Presets

## Scope

Corbina's Pipelines connector picker now exposes connection lifecycle filters
that match the `/workflows connections` search terms already supported by
Puffer Core and the TUI.

## Behavior

- The connector filter row includes `Repair`, `Active`, and `Idle` presets.
- `Repair` writes `repair` into the connector search box and surfaces
  connections that can be reconnected or repaired.
- `Active` writes `active` into the search box and surfaces connections whose
  consumer is running.
- `Idle` writes `idle` into the search box and surfaces connections without an
  active consumer.
- The presets reuse the existing indexed connection metadata; no daemon API
  contract changes are required.

## Test Coverage

The Pipelines Playwright fixture now models one active Telegram connection and
one idle Slack connection. The connector preset regression covers each new
preset's query value and result count so the GUI stays aligned with stable TUI
filter terms.
