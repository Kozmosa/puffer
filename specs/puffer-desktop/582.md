# Proxy Switch Empty List State

Updates Settings -> Network proxy switch behavior when no proxy endpoint exists.

- The proxy switch renders unchecked and disabled whenever the proxy list is
  empty, regardless of stale backend `enabled` state.
- Deleting the final proxy endpoint persists `enabled=false`, `selected=null`,
  and an empty proxy list.
- Enabling proxy routing with saved endpoints but no selected endpoint selects
  the first saved endpoint before saving.
- The save payload is normalized before `save_proxy_settings`, so the UI does
  not surface stale selected-proxy validation text for empty lists.
