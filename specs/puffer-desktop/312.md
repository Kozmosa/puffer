# 312. Solid PDF Preview Controls

## Summary

The Files PDF preview now uses a solid sticky control strip for zoom controls
and long-document status messages.

## Behavior

- Zoom controls and page-limit messages share one visible sticky strip at the
  top of the PDF preview.
- The strip and status chip use hard color fallbacks, borders, and shadows so
  they remain distinct in the desktop WebView.
- Zoom remains clickable after scrolling a long PDF preview.

## Compatibility

The preview still renders the first 20 pages, keeps the 50% to 200% zoom range,
and preserves the existing text fallback for renderer failures or searchable
PDF text.
