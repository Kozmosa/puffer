# 313. Browser Recording Hydrates Stale Local Tabs

## Summary

The Browser pane now prefers recorded agent browser frames when the daemon has
no live tab state, even if local storage contains stale saved tabs.

## Behavior

- Opening Browser for a session with only disconnected local tabs requests the
  session recording snapshot.
- Recorded agent tabs replace stale local placeholders and become the selected
  visible tab.
- The canvas, URL field, and tab title reflect the recorded agent browser
  action instead of staying on an empty or stale local page.

## Compatibility

Live daemon tab state remains authoritative. Local saved tabs are still used
when no recording frames are available.
