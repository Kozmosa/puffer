# Pipelines workflow action deletion

Corbina's Pipelines screen can now remove subscription-backed workflow actions
without asking the user to copy a terminal command.

## UX

- Monitor workflow rows show a `Delete` action next to pause/resume.
- File append workflow action rows show the same `Delete` action.
- The action calls `workflow_binding_delete` with the selected binding slug,
  applies the returned workflow snapshot, and reports success or failure in the
  existing Pipelines status line.
- While a workflow binding is being toggled, deleted, or created, binding row
  actions are disabled to keep snapshot mutations serialized.

The existing pause/resume behavior, append workflow creation form, connector
search filters, and monitor task panels keep their existing contracts.
