# Provider Settings Catalog Curation

- The Providers settings UI hides Cerebras, Groq, llama.cpp, LM Studio, vLLM, and Ollama cards.
- Google and Custom provider setup cards are appended when not already present in the daemon provider catalog.
- The curation is scoped to the Providers settings UI and does not change global provider routing catalogs used by onboarding or model selection.
