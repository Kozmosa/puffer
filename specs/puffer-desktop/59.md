# Desktop file symbol coverage

The desktop e2e harness now covers the Files panel language-service symbol
popup used while inspecting code.

Compatibility contract:

- The Playwright fake daemon implements `lsp_inspect` with the same operations
  map shape the desktop API expects.
- Moving the editor cursor onto an identifier sends the active file path,
  workspace cwd, line, and character to the daemon.
- Successful language-service results render the symbol popup with hover text,
  definition/reference locations, and a close control.

Regression coverage lives in `files-workspace-ui.spec.ts` alongside the file
save tests.
