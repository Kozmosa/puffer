# Terminal Close Target Labels

## Summary

Terminal tab close controls now expose the concrete terminal title in both the
accessible name and hover tooltip.

## Behavior

- Each close control keeps the existing accessible name shape, such as
  `Close Terminal 1`.
- The browser tooltip now matches that target-specific label instead of the
  generic `Close terminal` string.
- Closing behavior, in-flight close disabling, and retry behavior are unchanged.

## Tests

`terminal-ui.spec.ts` creates two terminal tabs and verifies each close control
has a target-specific title.
