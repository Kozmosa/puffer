# Playwright dev server launch

Desktop Playwright tests start the Vite development server without depending on
a globally installed `npm` binary.

Compatibility contract:

- The Playwright `webServer` command invokes the local Vite entrypoint through
  the same Node executable that is running Playwright.
- The command continues to serve `127.0.0.1:1420` so existing tests and app
  URLs remain unchanged.
- Existing servers are still reused outside CI.
- Test runs from environments with only a bundled Node runtime can cold-start
  the desktop UI.

Regression coverage is the Playwright cold-start path itself.
