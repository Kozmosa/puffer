# Pending Tool Card Surface

## Scope

Pending tool cards should use the same neutral card surface as other desktop
tool call UI.

## Change

- The `awaiting result...` panel now uses the standard background instead of a
  terminal-style dark surface.
- The pending shimmer is an inline muted/accent indicator so it fits inside
  both regular tool cards and collapsed Agent activity detail panels.
- ImageGeneration does not receive special-case rendering; it uses the shared
  pending tool card path.

## Compatibility

Terminal output cards keep their dark terminal surface. Only the generic
pending/no-output-yet tool state changes appearance.
