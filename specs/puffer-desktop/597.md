# Remove Browser Profile Settings

## Summary

Desktop settings no longer expose Chrome profile selection for Puffer browser
sessions.

## Behavior

The settings snapshot and config patch types no longer include
`browserChromeProfile`, and the UI no longer renders the browser profile picker
or account metadata. Mock settings data follows the same shape as the daemon API.

## Compatibility

The desktop app relies on the daemon-managed global Puffer browser profile for
browser-backed connectors. Existing settings screens continue to load because
the removed profile fields were optional client metadata rather than required UI
state.
