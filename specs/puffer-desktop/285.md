# 285. Browser DevTools Selected State

## Summary

Browser DevTools controls now expose their selected state through
`aria-pressed`, matching the visible active styling.

## Behavior

- The toolbar DevTools button reports `aria-pressed="false"` while the panel is
  closed.
- Opening DevTools changes the toolbar button to `aria-pressed="true"`.
- Console and Network view buttons report which DevTools view is active.
- Switching between Console and Network updates both buttons immediately.

## Compatibility

This is a semantic UI update only. It does not change daemon browser events,
DevTools event collection, tab state, or browser command routing.
