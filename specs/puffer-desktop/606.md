# Connector setup question images

The Settings connector setup modal now renders constrained markdown images in
connector question text. This covers connector-specific
setup prompts such as Telegram QR login, which are displayed by Settings rather
than the chat session question component.

Image sources are limited to `data:image/*;base64` and HTTP(S) URLs, while all
non-image text remains escaped Svelte content. Connector option previews remain
plain textual previews; QR login images are expected to live in the question
body.
