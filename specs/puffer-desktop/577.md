# 577 - Proxy Editor Modal Parity

Updates the Settings -> Network proxy editor to follow the Create connection modal pattern.

- Uses the shared `pf-modal` shell, focus trap, modal body, and footer button row.
- Removes editable proxy ID fields from Add and Edit flows.
- Creates proxy IDs automatically for new proxies and preserves existing IDs while editing.
- Validates host and port before saving, with the endpoint URI shown as the form preview.
