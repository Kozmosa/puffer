# Telegram Bot Connector Picker Coverage

## Summary

The Pipelines connector picker test daemon now includes the built-in
`telegram-bot` catalog row so Corbina covers bot setup selection as well as
personal-account Telegram login.

## Behavior

- The fake daemon reports `telegram-bot` with its deterministic
  `/connect telegram-bot telegram-bot` command.
- The connector row is searchable with `telegram bot`, shows events, proxy,
  no-trigger, and action metadata, and remains selectable for setup.
- Selecting the row stages the connect command without mutating the workflow
  trigger to a planned connection.

## Compatibility

This is additive desktop coverage for the existing catalog row. Existing
Telegram personal-account, Slack, email, Discord, and connection-trigger tests
are unchanged.
