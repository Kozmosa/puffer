# Wire settings keyboard shortcut

The desktop shell now implements the `Cmd/Ctrl + ,` shortcut that the Settings
shortcuts pane advertises.

Compatibility contract:

- The shortcut is ignored while onboarding is active so it cannot bypass login.
- Once in the workspace shell, `Cmd + ,` or `Ctrl + ,` opens Settings through
  the same screen-selection path as the sidebar.
- Existing Settings shortcut copy now describes the implemented command
  directly.

Regression coverage lives in `settings-ui.spec.ts`, pressing `Ctrl + ,` from
the workspace and checking that the General settings pane opens.
