# puffer-desktop New Agent Modal Escape

## Purpose

The New agent provider modal must follow the same keyboard cancellation behavior
as other desktop modals.

## Design

`NewSessionModal` now installs an Escape key listener while mounted. Pressing
Escape closes the dialog when it is not busy; during session creation, the modal
continues to ignore cancellation just like its close button.

## Logic

- Escape calls the existing `onClose` callback.
- The listener is removed when the modal unmounts.
- The behavior matches `WorkspacePicker` and `ConnectProjectModal`.

## Contracts

`files-workspace-ui.spec.ts` opens the New agent modal from the workspace board
and asserts that Escape removes the dialog.
