# Ignore stale directory picker responses

The Connect Project directory picker keeps only the most recent directory load.

Compatibility contract:

- Each `list_dir` request owns a monotonically increasing picker generation.
- Slow responses from older picker paths must not overwrite the visible path,
  directory rows, error state, or loading state from a newer path.
- Fast path changes through the picker text field and Go button remain usable
  while an earlier Browse-triggered load is still pending.
- The Tauri native directory chooser fallback continues to use the same
  in-modal picker in browser tests.

Regression coverage lives in
`apps/puffer-desktop/tests/files-workspace-ui.spec.ts`.
