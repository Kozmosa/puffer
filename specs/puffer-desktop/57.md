# Desktop agent detail coverage

The desktop Playwright harness now exercises more of the normal personal-agent
detail workflow instead of only browser-specific paths.

Compatibility contract:

- The fake desktop daemon can return session-level agent edits, git diffs, diff
  history, repo status overrides, and agent/git divergence data from
  `load_session_detail`.
- The Agent detail Diff tab must render the agent edit view, the git diff view,
  and the reconciliation view from daemon session detail data.
- Meta-clicking a top-level agent tab opens that tab in the side panel while the
  primary chat view remains visible.
- The Agent detail find overlay searches both the primary detail panel and side
  panel content, advances through matches, and restores original text when it
  closes.

Regression coverage lives in `agent-detail-ui.spec.ts`, using one mocked
session with transcript text, agent edit summaries, a working-tree diff, and a
deliberate agent/git divergence.
