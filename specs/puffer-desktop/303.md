# 296. Sidebar Current Page State

## Summary

The desktop sidebar now exposes the active primary screen to automation and
assistive technology instead of only marking it with `data-active`.

## Behavior

- The active primary navigation item reports `aria-current="page"`.
- Inactive primary navigation items omit `aria-current`.
- Switching from Project to Pipelines or Settings updates the exposed current
  page state together with the existing visual active state.

## Compatibility

This is a semantic shell update only. It does not change sidebar layout,
screen routing, active agent grouping, pinning, or resize behavior.
