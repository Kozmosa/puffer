# AskUserQuestion markdown images

The desktop `QuestionPrompt` renderer now supports constrained markdown image
syntax in question text and option previews. Text remains escaped Svelte
content, while image sources are rendered only for `data:image/*`, `https`, or
`http` URLs.

This lets connector setup prompts display generated QR codes inline, including
Telegram QR login previews, while preserving the existing answer and retry
behavior for choice, searchable choice, multi-select, and input questions.
