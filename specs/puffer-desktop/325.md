# 325. PDF Zoom Strip Visibility

PDF previews must make long-document page limits and zoom controls plainly
visible in the Files pane.

Contracts:

- The PDF control row is a full-width docked strip outside the scrollable page
  surface.
- Page-limit status text uses a high-contrast pill that is visually distinct
  from both the PDF preview background and the control strip background.
- PDF zoom exposes icon buttons, a visible range slider, and Ctrl/Command-wheel
  zoom so users can resize pages without hunting for tiny controls.
- Zoom updates already-rendered canvases in place and preserves the 20-page
  render cap for long PDFs.

Playwright coverage opens a 29-page wide PDF, verifies the status contrast,
uses the zoom slider, and checks that Ctrl-wheel zoom also changes the visible
zoom percentage without re-rendering extra pages.
