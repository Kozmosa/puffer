# 338. Deployment Secret Sync Feedback

Deployment secret controls must not expose inert sync actions.

Contracts:

- The Secrets pane `Sync` button is labeled as a secret sync action for
  assistive technology.
- Clicking `Sync` disables the control, reports `aria-busy=true`, and announces
  that the selected deployment's secrets are syncing with Vault.
- When the simulated sync completes, the live status reports the number of keys
  refreshed for that selected deployment and re-enables the button.
- Switching deployments clears stale sync status so the next environment does
  not inherit a previous environment's secret-sync feedback.

Playwright coverage opens Deployments, switches to Secrets, clicks the sync
control, verifies busy/live status, completion copy, and the restored enabled
state.
