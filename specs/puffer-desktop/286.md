# 286. Model Picker Provider Selected State

## Summary

Model picker provider buttons now expose their selected state through
`aria-pressed`. The provider switcher is labeled as a model-provider button
group.

## Behavior

- The active provider reports `aria-pressed="true"`.
- Inactive providers report `aria-pressed="false"`.
- Selecting another provider updates the pressed state immediately, including
  while its model list is still loading.
- Model option rows continue to use `aria-selected` inside the model listbox.

## Compatibility

This is a semantic update for the picker UI only. It does not change provider
filtering, lazy model loading, model selection, or turn request payloads.
