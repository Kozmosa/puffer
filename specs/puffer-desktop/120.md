# Session-Scoped Title Editing

## Scope

Agent detail supports inline session-title editing from the header. The edit
draft and saving flag are transient UI state for the currently selected
session, and must not carry across sidebar session switches.

## Change

- Agent detail now tracks the session id that owns the active title edit state.
- Switching to another session cancels title editing, clears any in-flight save
  flag, and resets the draft to the newly selected session's display name.
- A Playwright regression test starts editing one session title, switches to a
  second session through the sidebar, and asserts the title input disappears
  and the second session title is shown.

## Compatibility

Title editing behavior is unchanged within the same session. This only prevents
an unsaved title draft from one session being shown or saved while another
session is active.
