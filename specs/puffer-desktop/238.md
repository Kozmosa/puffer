# Puffer Desktop Update 238

## Change

- Removed the duplicate brand mark from the native titlebar.
- Kept the sidebar brand mark above Project as the only in-window Puffer mark.
- Granted the Tauri window dragging permission required by titlebar drag regions.

## Architecture

- `TitleBar.svelte` now renders a drag-only spacer instead of a logo container.
- The native window move behavior still uses Tauri's `data-tauri-drag-region`
  contract, but the desktop capability now explicitly allows
  `core:window:allow-start-dragging`.
- The shell chrome Playwright coverage verifies that the titlebar has no brand
  logo, the sidebar keeps exactly one logo, and the native drag permission is
  present in the Tauri capability file.

## Compatibility

- Web builds keep the titlebar hidden as before.
- macOS Tauri builds keep the 44px overlay-safe titlebar and traffic-light left
  inset, with a larger uninterrupted drag surface.
