# 362. Deployment Search Empty Detail

Deployment search must not leave a stale environment detail selected when the
filtered list has no matches.

## Contract

- A search with matching deployments still selects and displays the first
  matching environment.
- A search with zero matches shows the list empty state and a detail empty state.
- The detail empty state must not show the previously selected deployment name.
- Detail-only actions such as Open and Redeploy are unavailable while no
  filtered deployment is selected.
- Clearing the search restores the full deployment list and normal detail
  behavior.

## Verification

`tests/shell-chrome-ui.spec.ts` filters Deployments to a single Cloudflare
environment, then filters to a no-match query and verifies that the detail pane
shows `No deployment selected` with no stale deployment actions.
