# Sidebar workspace navigation exits drill-in views

The global sidebar Workspace item now clears workspace-local drill-in state
before showing the workspace screen. This makes the Workspace nav item behave
as a true return to the project/session board from agent detail or project
detail views.

The selected session is left intact for sidebar highlighting and remembered
session behavior, but `openAgentSessionId` and `openProjectId` are cleared so
the main stage renders the board instead of keeping the previous detail pane.
