# Sidebar accent color

## Design

The desktop sidebar now derives its selected, hover, avatar, resize, and pinned
icon accent styling from the global `--puffer-accent` variable that Appearance
sets on the document. Sidebar-specific CSS variables wrap that accent so the
sidebar can keep its own contrast levels while following the active Accent
choice.

## Contracts

The Appearance accent remains controlled by `data-accent` on the document root.
No Svelte state shape, settings persistence, or shell tweak contract changes are
required.

## Compatibility

The previous fixed violet sidebar colors are intentionally removed. Existing
sidebar layout, navigation behavior, resizing, and account/project rendering are
unchanged.

## Tests

`npm run test:sidebar-css` verifies that the sidebar no longer contains the
legacy fixed violet colors and that key sidebar rules reference
`--puffer-accent` through sidebar accent variables.
