# Pipeline Connector Action Chips

## Summary

The Pipelines connector picker now displays connector action slugs on matching
connection and connector rows.

## Behavior

- Connection rows show action chips from their connector template.
- Connector catalog rows show action chips directly from the connector template.
- Each row displays up to three action slugs.
- If the current search terms match action slugs directly, matching actions are
  prioritized before the default first three actions.
- Rows with additional actions show `+N actions` after the visible slugs.
- Search behavior continues to index action slugs, so a connector found by an
  action term now visibly explains why it matched.

## Compatibility

Trigger selection, non-trigger setup-row selection, and deterministic
`/connect` command display are unchanged. The action chips are additive metadata
inside the existing connector tag area.
