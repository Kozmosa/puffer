# 326. Sub-agent Activity Labels

## Summary

Desktop chat activity treats `spawn_agent`, `wait_agent`, `send_input`,
`close_agent`, `resume_agent`, and legacy sub-agent spellings as sub-agent
actions.

## Behavior

- Activity summaries count these tools as sub-agent usage.
- Expanded action rows show readable labels such as `Spawn sub-agent` instead of
  raw tool IDs.
- Tool detail cards surface role, model, effort, status, and prompt/message
  metadata for sub-agent actions.

## Verification

- Playwright seeds a `spawn_agent` tool call and verifies the activity summary,
  action row, and detail panel show sub-agent copy without exposing the raw tool
  ID.
