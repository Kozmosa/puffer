# Diff Timeline Activity

## Context

Desktop session details can include `diff_snapshot` timeline entries. The chat
view had renderer support for diff activity cards, but the row derivation
filtered diff entries before they reached that renderer.

## Change

- Keep normalized `diff` timeline items in the chat activity row builder.
- Treat diff-only timelines as started conversations for empty-session provider
  switching rules.

## Compatibility

The side-panel Diff tab and action-history panes are unchanged. This only makes
existing timeline diff snapshots visible in the transcript activity stream.

## Verification

- Added a Playwright regression where a session timeline contains a
  `diff_snapshot` between a user prompt and final assistant message, and the
  chat activity summary remains visible.
