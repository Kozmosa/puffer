# Image Settings Media Directory

## Scope

The desktop image generation settings modal and "Open folder" command should
refer to the canonical workspace image media directory.

## Change

- The settings modal displays `<session cwd>/.puffer/media/images`.
- `open_image_dir` creates and opens `.puffer/media/images` for the session
  cwd.
- The Tauri command keeps rejecting non-absolute cwd values.

## Compatibility

The desktop app no longer displays, creates, or opens
`.puffer/workflows/images` for image generation settings.
