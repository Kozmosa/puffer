# Provider Connection Modal Layout

- The default routing area is now a card-style panel that keeps the provider and model selectors together with the save action.
- The Save default button height matches the adjacent select controls, so the row aligns visually.
- Provider cards now hide credential configuration controls and only expose a single Connect or Manage connection action.
- Credential imports, OAuth reconnect, and API-key entry now live inside a provider-scoped modal opened from the card action.
- The modal keeps the existing provider colors, credential handlers, loading states, and error behavior while reducing visual noise in the provider grid.
