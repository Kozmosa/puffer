# Corbina Workflow Draft Commands

## Summary

The Pipelines connector picker now exposes the same deterministic workflow
draft commands shown by the terminal `/workflows` dashboard.

## Contract

- Trigger-capable connector and connection rows index `draft`, `workflow`,
  `new`, and the rendered `/workflows new <connection>-workflow
  <connection>` command.
- The connector filter bar includes a `Draft` preset that narrows results to
  event-capable workflow trigger sources.
- Draft icon tooltips use the exact `/workflows new ...` command for the
  selected connection or connector-derived connection slug.
- The selected connector setup panel shows a read-only workflow draft command
  for valid trigger-capable connector connection names, with a copy action.
- Setup-only connectors still hide draft commands and remain available through
  connect/setup search paths.

## Compatibility

This is a Corbina UI-only change. Workflow creation remains local until the
user saves the draft, and existing daemon workflow snapshot payloads remain
compatible.
