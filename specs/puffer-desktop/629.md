# Puffer Desktop Update 629

## Summary

Linux CEF release builds now restore Chromium lifecycle hooks that CEF depends
on when embedding Chrome runtime code.

## Design

- `scripts/release.sh` adds a Chrome lifecycle compatibility patcher for CEF
  release builds.
- The patcher restores the browser cleanup hook expected by CEF shutdown.
- It restores the request-aware external protocol interception hook used by CEF
  URL loader proxying.
- It restores renderer lifecycle forwarding for render-thread connection and
  DevTools attach/detach notifications.

## Compatibility

The hooks are idempotent and preserve Chrome's default behavior when CEF does
not handle the event. This keeps the current Chromium Tintin checkout compatible
with the CEF 7680 patch set without changing the Tauri integration contract.
