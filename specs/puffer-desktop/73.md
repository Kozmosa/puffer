# Clear composer draft on session switch

Desktop composer drafts are scoped to the currently selected session.

Compatibility contract:

- Switching from one session to another clears unsent composer text before the
  next session can send a turn.
- Same-session transcript reloads and live turn updates keep the composer
  lifecycle unchanged.
- An empty composer remains non-submittable, including after keyboard Enter.
- The session switch still scrolls the newly selected transcript to its start.

Regression coverage lives in
`apps/puffer-desktop/tests/chat-session-ui.spec.ts`.
