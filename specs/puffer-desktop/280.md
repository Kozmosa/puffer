# Agent Detail Side Panel Deduplication

## Summary

When a user opens a tab in the side panel and later switches the primary agent
view to that same tab, the duplicate side panel now closes.

## Behavior

- Meta/Ctrl-click still opens allowed side-panel tabs such as Diff.
- Normal tab clicks still switch the primary agent detail body.
- If the primary tab becomes the same tab already shown in the side panel, the
  side panel is removed to avoid duplicate content and wasted width.
- Effectful tabs remain protected by the existing side-panel allowlist.

## Tests

`agent-detail-ui.spec.ts` opens Diff in the side panel, then switches Diff into
the primary view and asserts that the duplicate side panel is gone.
