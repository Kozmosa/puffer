# 281. Agent Detail Tab State Semantics

## Summary

Agent detail pane switches now expose their selected state through
`aria-pressed` and use explicit `type="button"` controls. The same contract
applies to the Diff pane's Agent, Git, and Agent/Git source switches.

## Behavior

- The Chat, Diff, Terminal, Files, and Browser controls remain regular buttons
  so existing click and modifier-click behavior is unchanged.
- Only the active detail pane reports `aria-pressed="true"`.
- Diff source buttons follow the same active-state contract.
- The controls live in labeled button groups for assistive technology.

## Compatibility

This is a semantic and accessibility update only. It does not change tab
ordering, side-panel routing, Browser/Terminal side-panel suppression, or Diff
data rendering.
