# 295. Pipeline Graph Node Selected State

## Summary

The pipeline editor graph now exposes which agent node is selected instead of
only showing that state through the visual halo.

## Behavior

- Agent graph nodes report `aria-pressed="true"` when selected.
- Non-selected graph nodes report `aria-pressed="false"`.
- Clicking another agent node updates the exposed state and the selected-agent
  inspector fields together.

## Compatibility

This is a semantic UI update only. It does not change workflow graph layout,
edge rendering, provider defaults, or daemon workflow contracts.
