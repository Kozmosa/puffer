# 293. Deployment Detail Tab Semantics

## Summary

The deployment detail section now exposes its pane switcher as an actual tab
control instead of unlabelled active buttons.

## Behavior

- The detail switcher reports `role="tablist"` with a stable label.
- Each deployment detail tab reports `role="tab"` and `aria-selected`.
- The active tab remains tabbable while inactive tabs use roving tabindex.
- Left/right/up/down arrows move between adjacent tabs.
- Home and End jump to the first and last deployment detail tab.
- The visible pane reports `role="tabpanel"` and is labelled by its active tab.

## Compatibility

This only changes tab semantics and keyboard navigation. It does not change the
selected deployment, mock deployment data, or any deployment pane content.
