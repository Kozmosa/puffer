# Browser Address Submit

The Browser pane address field now owns an explicit Enter-key navigation path
instead of relying only on native form submission. This keeps keyboard
navigation reliable in embedded/browser-automation surfaces that update the
input value but do not synthesize a form submit from Enter.

The toolbar also exposes a compact Go icon button beside the address field.
It uses the same `submitUrl` path as Enter, honors the existing address enabled
state, and preserves the current daemon `browserNavigate`/`browserOpen`
contracts.
