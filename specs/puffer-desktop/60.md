# Desktop focused input shortcuts

The agent detail global find shortcut no longer steals focus from embedded
editing surfaces.

Compatibility contract:

- `Ctrl/Cmd+F` opens the agent-wide find overlay only when focus is outside an
  editable control.
- Files editor textareas keep focus when users invoke their normal find
  shortcut, so editing workflows are not interrupted by the agent overlay.
- Terminal input keeps the same shortcut for shell/readline use instead of
  opening the agent-wide overlay.
- The find overlay remains keyboard controlled while focus is inside its own
  controls.

Regression coverage lives in `files-workspace-ui.spec.ts` and
`terminal-ui.spec.ts`.
