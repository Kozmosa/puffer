# 401. PDF Status Text Targets The Banner

PDF page-limit status text is rendered directly inside the status banner rather
than in a nested text-only span. This keeps text locators, accessibility
snapshots, and visual style checks anchored to the visible high-contrast banner
instead of a transparent child node.

The page-limit label remains a separate pill so the long-document warning stays
obvious while existing document-format preview tests continue to inspect the
actual banner background and border.
