# Scoped Title Save Completion

## Summary

A delayed title save from one session no longer closes or disables title editing
in another session.

## Design

`AgentDetail.svelte` now tracks a title-edit generation. Starting an edit,
canceling an edit, or switching sessions advances the generation. A pending
save only clears the editing state and saving flag when it returns to the same
generation and session that started it.

## Contracts

- Normal title saves still close the active title editor.
- A delayed rename response must not overwrite a switched session's title.
- A delayed save from a previous session must not cancel an edit that the user
  started in the currently selected session.

## Verification
