# Session-Scoped Agent Find

## Scope

Agent detail has a local find overlay for searching the visible chat/detail view.
The active query and highlighted marks belong to the currently selected session.

## Change

- Agent detail now closes find and clears search marks whenever the selected session changes.
- This prevents a search query from one session from staying open and highlighting another session after sidebar navigation.
- A Playwright regression test opens find in one session, searches text, switches to another session, and asserts the find overlay and marks are gone.

## Compatibility

Find behavior is unchanged within the same session. Closing find on session change
mirrors the existing draft/title/file-link state reset behavior and avoids stale
DOM marks in the new session.
