# Verified Skill Gate Toasts

Verified Skill Gate timeline items render inside the surrounding agent row as
compact gate toasts instead of regular system messages. The renderer
recognizes both live `lambda-gate` items and persisted `Verified Skill Gate
...` system messages, preserving the timeline event while avoiding the bulky
SYS/ERR block treatment and not interrupting chat grouping.

Live gate events now store concise human text rather than host/concrete JSON
payload dumps. Detailed structured metadata remains on the corresponding tool
invocation for future gate-aware UI.

Validation:
- `npm run check`
