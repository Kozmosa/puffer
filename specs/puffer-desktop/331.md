# 331. PDF Preview Controls Visibility

PDF previews must make long-document status and zoom controls visually distinct
enough to use without guessing.

## Contract

- The PDF control strip includes a visible `Zoom` label, icon buttons, a range
  slider, and a separate numeric zoom value.
- The long-document status such as `Showing first 20 of 29 pages.` uses a
  high-contrast amber badge that differs from both the file preview background
  and the control strip background in light and dark themes.
- Zoom controls remain normal pointer targets above the page scroller, so the
  slider and buttons can be clicked after the PDF has been scrolled.
- Existing PDF rendering behavior still caps long previews at the first 20
  pages and updates rendered canvas dimensions without re-rendering pages.

## Coverage

Playwright opens a 29-page wide PDF in dark theme, verifies the explicit zoom
label and amber status badge, hit-tests the slider and zoom button, then changes
zoom through the slider, button, and Ctrl-wheel without rendering more than 20
pages.
