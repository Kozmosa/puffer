# 591 - Network Proxy Bypass Validation

Updates Settings -> Network bypass editing behavior.

- The Bypass textarea keeps local user edits until a new saved snapshot arrives.
- Save bypass uses the standard default button style.
- Save bypass validates normalized comma- and newline-separated entries before
  sending a daemon request.
- Valid entries include hosts, IP addresses, and IPv4/IPv6 CIDR ranges.
- Unsupported wildcard, malformed CIDR, and backslash entries fail locally and
  leave the persisted proxy settings unchanged.
