# Replay-safe assistant text deltas

Assistant `text-delta` replay is now idempotent when the Desktop UI already
has live streamed text for the same turn:

- The app tracks the replayed text prefix per `turnId`.
- If the retained live assistant row already starts with the replay prefix, the
  replayed delta is ignored.
- If replay catches the UI up beyond the retained text, only the missing suffix
  is appended.
- Fresh non-replay turns still clear live stream rows and reset replay tracking.

This prevents reconnect replay from rendering duplicated assistant text while
still preserving legitimate repeated chunks such as `ha` + `ha`.
