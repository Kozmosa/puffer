# 363. PDF Narrow Light Controls

PDF previews must keep the long-document limit badge and zoom controls obvious
in narrow light-theme file previews.

## Contract

- The `Showing first 20 of N pages.` badge uses high contrast against its own
  background and does not visually blend into the PDF preview background.
- Narrow previews keep the zoom toolbar and range input visible and pointer
  targetable.
- Zoom buttons and the range input update the rendered canvas size immediately.
- The long-document cap remains 20 rendered canvases.

## Verification

`tests/files-workspace-ui.spec.ts` opens a 29-page wide PDF in a narrow
light-theme Files preview, checks strong badge contrast, verifies the zoom
button and slider receive pointer hits, then confirms both controls update the
visible zoom percentage and canvas width.
