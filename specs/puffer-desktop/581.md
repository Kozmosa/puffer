# Proxy List Delete Action

Adds a delete action to each Settings -> Network proxy list item.

- Proxy rows show a `Delete` button with the shared trash icon next to Test and
  Edit.
- Deleting a saved endpoint persists the remaining proxy list through the
  existing `save_proxy_settings` request.
- If the deleted endpoint was selected, the first remaining endpoint becomes
  selected. If no endpoints remain, proxy routing is disabled and `selected`
  becomes `null`.
- A matching last connectivity result is cleared when its proxy endpoint is
  deleted so stale status text is not carried forward.
