# 403. Live Sidebar Session Event Subscriptions

Running sidebar agents keep a lightweight session-event subscription even after
the user opens another session. Completion/error events for the hidden live
session now clear its sidebar overlay and cached turn runtime state without
waiting for a separate workspace session-list refresh.

The selected session continues to use the primary session subscription. Live
sidebar subscriptions ignore events while their session is selected, which keeps
stream handling single-sourced and avoids duplicate transcript updates.
