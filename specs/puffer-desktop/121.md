# Session-Scoped Activity Expansion

## Scope

Conversation activity groups are collapsible summaries for tool calls,
intermediate assistant messages, and diffs. Expanded groups and selected
activity children are local view state for the currently selected session.

## Change

- Conversation view now clears expanded activity groups and selected activity
  children whenever the selected session changes.
- This prevents one session's expanded activity group from automatically
  opening another session's group when backend transcript ids overlap.
- A Playwright regression test creates two sessions with the same transcript
  item ids, expands activity in the first session, switches to the second, and
  asserts the second session starts collapsed.

## Compatibility

Draft reset and scroll-to-top behavior on session change are unchanged. Activity
expansion still persists while staying within the same session.
