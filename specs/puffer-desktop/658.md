# Captcha solver readiness status

The Browser settings solver UI now separates solver selection from readiness.
A selected solver only shows as ready when captcha solving is enabled and an API
key is already stored or has been entered in the draft form.

The Extensions subtab uses the same readiness rule for built-in solver
extensions, while non-selected solver extensions remain listed as available
solver extensions. This prevents an enabled-but-unconfigured solver from
appearing active.
