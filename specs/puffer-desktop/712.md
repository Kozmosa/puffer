# Contacts Duplicate Identity Save Coverage

## Design

The desktop fake daemon mirrors daemon contact-save identity replacement:
saving a contact absorbs existing saved contacts whose normalized connector IDs
overlap the new record and preserves their non-overlapping connector IDs. The
Contacts UI regression covers the modal create flow and keeps the newly saved
replacement selected.

## Compatibility

This changes only desktop test support and coverage. The production desktop RPC
payload is unchanged and still sends the user's trimmed contact IDs to the
daemon.
