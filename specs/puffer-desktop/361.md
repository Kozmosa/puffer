# 361. Ask Puffer IME-Safe Submit

Deployment Ask Puffer uses the same Enter-to-send rules as the main chat
composer for users typing with an IME.

## Contract

- Pressing Enter while the textarea is composing text must not submit the draft.
- Browsers that report IME Enter as key code 229 must also leave the draft in
  place.
- Plain Enter still sends the prompt, clears the draft, and appends the local
  deployment investigation turn.
- Shift+Enter remains reserved for multiline text.

## Verification

`tests/shell-chrome-ui.spec.ts` opens Deployments, sends prompts from the button
and plain Enter, then dispatches IME composition Enter events into the Ask
Puffer textarea and verifies that the draft remains unsent.
