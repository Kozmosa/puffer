# Tauri session-store dependency

- The desktop Tauri crate now declares its direct dependency on `puffer-session-store`.
- DTOs use `MessageActor` from the shared session-store crate, so the dependency belongs in `src-tauri/Cargo.toml` instead of relying on transitive workspace crates.
- The change keeps the Tauri crate independently checkable with `cargo check --manifest-path apps/puffer-desktop/src-tauri/Cargo.toml`.
