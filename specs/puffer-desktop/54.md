# Stabilize Codex MCP smoke test

The desktop Playwright smoke suite includes a Codex CLI check for the built-in
Playwright MCP configuration. That command can take longer during a full
parallel browser run than during an isolated invocation, so the test now grants
the command a larger timeout budget.

Compatibility contract:

- The assertion still requires `codex mcp list` to report the Playwright MCP
  server, `npx`, and `@playwright/mcp`.
- The test change does not alter application code or MCP configuration.
- The larger timeout only reduces false negatives from process startup or
  runner contention.

Regression coverage is the existing `codex-playwright-mcp.spec.ts` smoke test.
