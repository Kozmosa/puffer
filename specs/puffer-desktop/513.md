# New Relic Connector Catalog UX

The Corbina/Puffer Desktop Pipelines connector catalog now exposes
`newrelic-webhook` with the same setup-only serve metadata returned by the
daemon.

## Search And Selection

- The all-connectors count increases to 25.
- Searches for `serve`, `serve webhook`, `setup-only webhook`, and
  `new relic alert webhook` include `newrelic-webhook`.
- The selected connector detail pane exposes `skill:newrelic-webhook` and the
  deterministic command `/connect newrelic-webhook newrelic-webhook`.

## Runtime Contract

The desktop fixture keeps New Relic as setup-only: no workflow trigger, no
proxy runtime, no connector actions, and `runtime_hints = ["serve"]`.
