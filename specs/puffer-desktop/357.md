# 357. Deployment Row Action Menus

Deployment Memory and Secrets rows must not render inert "More" controls.

- Memory row More buttons expose per-note actions with unique accessible names.
- Memory actions can pin a note locally or queue it as Ask Puffer context, and
  both actions produce visible status feedback.
- Secrets row More buttons expose per-key actions with unique accessible names.
- Secret actions can queue or clear rotation state and queue access audits with
  visible status feedback.

Playwright coverage opens both row menus, runs the actions, and verifies the row
state and status messages that prove the controls are not dead.
