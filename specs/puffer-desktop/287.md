# 287. Model Picker Trigger Expanded State

## Summary

The model picker trigger now exposes its popup relationship and open state.

## Behavior

- The trigger reports `aria-haspopup="listbox"`.
- The trigger reports `aria-expanded="false"` while the picker is closed.
- Opening the picker changes the trigger to `aria-expanded="true"`.
- Closing the picker with Escape returns the trigger to
  `aria-expanded="false"` and keeps the existing focus behavior.

## Compatibility

This is a semantic UI update only. It does not change provider filtering,
lazy model loading, model selection, or composer turn payloads.
