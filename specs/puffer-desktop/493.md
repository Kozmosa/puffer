# Pipeline Task Search Preset

## Scope

Corbina Pipelines now treats connector monitor tasks as first-class search
matches in the connector picker area.

## Behavior

- The connector filter preset row includes `Tasks`, which writes the stable
  query `monitor task`.
- The existing connector search input filters monitor tasks with the same
  all-terms matcher already used by connectors and connections.
- When active monitor tasks exist, Pipelines shows a separate
  `Monitor task search results` summary with filtered and total task counts.
- Existing connector and connection result summaries remain unchanged, so
  connector-specific scan patterns do not need to parse task counts.

## Compatibility

The task preset and summary are additive. Existing connector setup, repair,
monitor startup, task action, and task ignore flows keep their current command
contracts.
