# Minimum Width Navigation

The desktop shell must keep primary navigation available at the configured
Tauri minimum window width. Since the native window can shrink to 720px wide,
the CSS breakpoint that hides the entire sidebar must start below that value.

At 720px the sidebar may remain compact and icon-only, but Workspace, Pipelines,
and Settings must still be reachable. Narrower web preview widths can continue
to hide the sidebar for mobile-style layouts.

Icon-only sidebar buttons must keep explicit accessible names and tooltips so
automation, keyboard users, and assistive technology can still target primary
navigation when the visual text is hidden.
