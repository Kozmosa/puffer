# 233. Workspace Project Row Collapse

## Summary

Workspace project rows now include an explicit expand/collapse control for the
session strip.

## Behavior

- Each project row starts expanded.
- The row header exposes an accessible collapse button with `aria-expanded`.
- Collapsing hides the session cards and new-agent tile while keeping project
  counts and row actions visible.
- Expanding restores the session list without navigating away from the
  workspace board.

## Compatibility

This is a local desktop UI behavior. It does not change daemon protocols,
session metadata, or project grouping.

## Tests

`workspace-ui.spec.ts` covers collapsing and expanding a project row and
asserts that session cards hide and return while project counts remain visible.
