# Contacts Workspace

- The sidebar Contacts tab replaces the previous Telegram-only Relationships entry
  and loads saved contacts plus ranked connector candidates through daemon contact
  RPCs.
- Users can create, edit, delete, and infer contacts. A saved contact groups multiple
  connector contact ids under one user-authored name and description.
- The Tasks configuration dialog loads saved contacts and connector candidates, keeps
  existing monitor scopes when editing, and expands selected saved contacts into the
  underlying contact ids before calling `task_monitor_create`.
- Empty monitor contact selection is displayed and sent as all contacts. Active
  monitor rows and filter details show whether the monitor is all-contact or scoped.
- The fake daemon implements contact RPCs and task monitor creation with contact id
  persistence so Playwright tests can exercise the desktop flow.
