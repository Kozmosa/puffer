# Terminal Split UTF-8 Regression

## Summary
- The desktop Terminal Playwright coverage now exercises PTY output whose UTF-8 bytes are split across daemon frames.
- The regression checks that multibyte text renders as intended and does not produce replacement glyphs.

## Compatibility
- This is a test-only guard for the existing PTY byte transport.
- The daemon protocol remains base64 byte frames; xterm remains responsible for incremental UTF-8 decoding.
