# Desktop Connector Catalog Without HTTP Ingress

## Change

The desktop test daemon and pipeline connector picker coverage no longer expose
HTTP ingress connector presets.

## Contract

- The fake daemon connector catalog contains the ten live connectors only.
- Connector search denominators use `10/10` and serve filtering returns the
  remaining serve-mode bot connectors.
- Playwright coverage no longer stages `/connect *-webhook` setup commands.

## Compatibility

Pipeline workflow drafting, append binding, connector setup command execution,
and monitor task flows continue to use the remaining connector catalog entries.
