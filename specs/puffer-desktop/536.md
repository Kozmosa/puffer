# Lambda Skill Library Settings

## Scope

Desktop Settings now has a Skills pane for configuring external verified Lambda
Skill libraries.

## Change

- The Settings navigation includes `Skills`.
- The pane shows manifest directories, current manifests, Lambda Skill doctor
  readiness, and warning lines from the daemon.
- Users can save workspace or user scoped library manifests with root,
  compiler, generated descriptor, host catalogue, allowed tools, global host
  bindings, per-skill host bindings, and model invocation toggles.

## Compatibility

The desktop UI writes the same YAML manifests that TUI users can create by
following `/skills config`. It does not copy or vendor skill libraries, and it
does not bypass the Lambda Skill readiness or host-call gate.
