# Puffer Desktop Update 239

## Change

- Added a draggable resize handle to the desktop sidebar.
- Persisted the sidebar width as a renderer-local shell tweak.
- Added shell chrome coverage for mouse resizing and local persistence.

## Architecture

- `Tweaks` now includes `sidebarWidth`, clamped between 220px and 420px.
- `Sidebar.svelte` owns pointer and keyboard resize interactions and reports
  clamped widths through `onResize`.
- `App.svelte` persists width through the existing tweak save path rather than
  workspace or session metadata.

## Compatibility

- Collapsed sidebar mode remains 56px wide and hides the resize handle.
- Existing tweak payloads migrate by falling back to the 248px default width.
