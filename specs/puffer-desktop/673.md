# puffer-desktop Update 673: Media Settings Select Styling

## Summary

Image and video generation settings dropdowns now reuse the same shared input
primitive as the Settings page Add server form.

## Frontend Contract

- All `MediaSettingsModal` select controls use the global `sc-input` class.
- The modal keeps its single-column field layout while inheriting Add server's
  select height, padding, border token, radius, background, font size, and
  shadow.
- No media settings persistence, capability loading, or save payload behavior
  changes.

## Tests

- Svelte checking validates the updated markup and scoped CSS.
- Existing media settings Playwright coverage continues to verify the dropdown
  labels and save behavior.
