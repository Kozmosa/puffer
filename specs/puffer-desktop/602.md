# Browser Setup Modal Polish

- Browser-backed connector setup uses a shorter, wider modal that gives most space to the embedded browser while keeping a compact right-side action rail.
- Action-only setup steps render as guidance text instead of bordered question cards.
- Browser setup actions live in the right rail so the dialog no longer has a wide empty footer under the browser.
- Embedded browser chrome is tightened inside connector setup to leave more vertical space for the actual login page.
