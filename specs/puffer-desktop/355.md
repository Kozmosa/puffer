# 355. Deployment Provider Settings

Deployment provider rows must not expose inert settings controls.

- Each provider settings button targets the specific integration by name.
- Activating the button opens an inline settings form for that provider.
- Saving the form updates the current deployment's provider kind, connection
  note, and status in local deployment state.
- Add-provider state and edit-provider state are mutually exclusive, so opening
  one form cannot leave the other form active.

Playwright coverage opens the Deployments Providers tab, edits the OpenAI
integration status and connection note, and verifies the selected provider card
and pane status update after saving.
