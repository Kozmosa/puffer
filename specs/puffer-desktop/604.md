# Browser Connector Setup Copy

Desktop connector setup treats `gcal-browser` as a browser-backed connector in
the same UI family as `gmail-browser`.

- Action-only browser prompts use the concise `Continue` submit label.
- Setup status text names Google Calendar explicitly for sign-in and account
  selection states.
- No new modal shell or connector-specific browser component was added; the
  existing browser setup modal continues to render daemon-provided browser
  sessions.
