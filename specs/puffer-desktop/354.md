# 354. Detail Header Title Width

Agent detail headers must not pre-truncate session titles while there is still
available horizontal room in the desktop window.

- The header identity block grows into the remaining space between navigation,
  status, tab controls, and the close button.
- Title text still uses CSS ellipsis when the header truly runs out of space,
  but there is no fixed 420px cap on wide layouts.
- The visible primary and secondary title spans expose the complete value in
  their `title` attribute for hover inspection.

Playwright coverage opens a long-titled session in a wide viewport, verifies the
full title remains in the DOM and tooltip, and asserts the header title region
expands beyond the old cap before ellipsis can apply.
