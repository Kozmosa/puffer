# 337. Deployment History Logs

Deploy history rows must not expose inert log icon buttons.

Contracts:

- Each deploy history row labels its log action with the deploy id.
- Clicking the log action opens a deploy-output panel for that exact run.
- The selected log action reports `aria-pressed=true` while its log panel is
  open.
- The log panel summarizes checkout, build, health, and traffic/rollback state
  using the selected deployment and deploy run metadata.
- Closing the log panel removes it without changing the selected deployment or
  detail tab.

Playwright coverage opens the Deploys tab, clicks the first run's log button,
checks the deploy-specific log output and pressed state, then closes the panel.
