# Mini Floating Window with Smart Screen Placement

- A global shortcut (`Cmd/Ctrl+Shift+Space`) summons a small (~1/10 screen, ChatGPT-mini style) always-on-top, undecorated companion window; the same chord toggles it away.
- On summon the desktop backend screenshots the main display and runs a local cold-region detector - a downscaled edge-activity map plus a summed-area table - to find the emptiest target-sized rectangle, then positions the window there in physical pixels (correct on retina) instead of always centering. The menubar strip and screen edges are reserved.
- When capture is unavailable (non-macOS, or `screencapture` fails) it falls back to a centered ~1/10-screen placement.
- The mini window is a launcher: it does not stream the conversation itself. Submitting emits `puffer://mini-submit`; the main window focuses and runs the prompt through the normal create-session-if-needed + submit path, then the mini window hides. It also dismisses on blur or `Esc`.
- A reference prototype with an optional vision-model refinement for the semantic-cold case lives in `scripts/mini-placement.py`.
