# Puffer Desktop Update 628

## Summary

Linux CEF release builds now patch Chromium's permission prompt API drift before
compiling CEF.

## Design

- `scripts/release.sh` adds an idempotent permission prompt compatibility
  patcher to the CEF release setup.
- The patcher restores CEF's permission prompt factory hook in Chromium's
  permission prompt factory.
- CEF delegate decisions now set prompt options separately before calling the
  current zero-argument `Accept`, `Deny`, `Dismiss`, and `Ignore` methods.

## Compatibility

This preserves CEF's embedder permission-prompt override behavior while matching
the current Chromium permissions delegate contract used by the local Tintin
checkout.
