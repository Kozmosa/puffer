# Native CEF Hit Testing

- Native CEF mouse focus now resolves the AppKit hit-tested frontmost view at
  the event point and maps that view's ancestor chain back to a visible browser
  slot.
- Overlapping slots no longer depend on `g_slots` map iteration order; a covered
  browser slot is not selected when another view is above it.
- Slot visibility, closing state, and content-view ownership checks are still
  enforced before focusing the embedded CEF browser.
