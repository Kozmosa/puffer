# Pipeline Connector Planning

## Summary

The Pipelines editor connector catalog is now actionable even before a
connection exists: users can search for a connector template, select it, and
see the deterministic `/connect` command needed to authenticate it.

## Behavior

- Connector template rows are selectable buttons in the catalog.
- Selecting a connector switches the draft workflow trigger to the first
  existing connection for that connector, or to the connector's suggested
  connection slug when no connection exists.
- The selected connector row remains highlighted.
- The editor renders the exact `/connect <connector> <connection>` command for
  the selected connector.
- The connection dropdown keeps planned, not-yet-authenticated connection slugs
  visible instead of rendering an empty select value.

## Compatibility

Existing connection-row selection behavior is unchanged. The new connector
fields from the daemon are optional in the TypeScript model and fall back to
`/connect <connector_slug> <connector_slug>` for older daemon snapshots.
