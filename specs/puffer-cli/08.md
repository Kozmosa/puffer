# Benchmark web tool permission coverage

The unattended benchmark runner uses a curated tool surface rather than the
full interactive tool registry. That surface intentionally includes `WebSearch`
and `WebFetch` so benchmark tasks can inspect public documentation or remote
context when the task requires it.

Compatibility contract:

- The default benchmark permission file marks curated tools as `allow`.
- Non-curated interactive tools such as `AskUserQuestion` remain `deny`.
- Explicit `--deny-tools` entries override the curated default for any listed
  tool, including web tools.

Regression coverage lives in the `benchmark_tool_permission_uses_curated_allowlist`
unit test.
