# Workflow tmux regression

- The tmux agent-loop integration suite now covers workflow action validation
  through the real TUI.
- The scripted model calls `WorkflowValidate` with step-style `write_file`
  YAML and verifies the follow-up provider turn receives a normalized
  `file_append` tool result.
- The same test inspects the first provider request and asserts
  `WorkflowCreate` advertises an object input schema without a top-level
  `anyOf`, preserving provider compatibility for model-visible tool schemas.
