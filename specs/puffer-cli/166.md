# Browser CDP Key Mapping

- CDP key input now derives virtual key codes for common punctuation, `Space`,
  navigation keys, function keys, numpad keys, and modifier codes in addition
  to `Key*` and `Digit*`.
- Named `Space` input is normalized to the browser key value `" "` and sends a
  printable space text payload on key-down events.
- Printable punctuation sent without a DOM `code` is canonicalized to the
  matching physical key code before dispatch so screencast fallback input stays
  closer to real browser keyboard events.
- Existing `windowsVirtualKeyCode` and `nativeVirtualKeyCode` parity is
  preserved; both are populated from the derived virtual key when available.
