# Multi Provider Video Capability Tests

Date: 2026-06-09

## Scope

Daemon media capability tests must tolerate multiple bundled executable video
providers.

## Behavior

- `list_media_capabilities` tests assert that the requested provider capability
  is present and available instead of asserting that it is the only video
  capability.
- The Relaydance override fixture uses the same Seedance 2.0 duration,
  resolution, and ratio options as bundled resources.

## Compatibility

The daemon RPC response shape is unchanged. This only updates test expectations
for the expanded provider catalog.
