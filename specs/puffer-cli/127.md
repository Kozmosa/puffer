# Gmail Browser Account Selection

- Gmail browser setup only attaches browser metadata to sign-in prompts; account selection is a normal connector question after discovery completes.
- Account selection asks for logged-in Gmail identities, not arbitrary emails visible in the mailbox.
- Account discovery reads Google account chooser rows and Gmail account/profile attributes, avoiding broad Gmail body-text scraping that can pick up message senders.
- Account chooser text nodes are separated before email matching so display names cannot be concatenated into false account emails.
