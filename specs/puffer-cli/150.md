# Auto-title language and term preservation

Daemon auto-title generation now asks the title model to preserve the user's
language and user-supplied names instead of translating or replacing them with
generic project wording.

The title prompt keeps the concise JSON contract, adds examples for named bug
reports and non-English input, and tells the model to preserve project names,
file names, package names, quoted terms, and unusual proper nouns exactly.

Generated titles are also validated against the first user message. If a model
returns a generic title that drops most distinctive user terms, Puffer falls back
to the deterministic first-message title rather than saving a misleading
translation or generic task name.
