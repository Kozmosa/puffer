# Task Agent Default Models

- Background task agents with no explicit monitor model use a provider-family task default instead of the general coding default.
- OpenAI task agents prefer `openai/gpt-5.4-mini` when the model is registered.
- Anthropic task agents prefer `anthropic/claude-haiku-4-5-20251001` when the model is registered.
- If the selected provider is not OpenAI or Anthropic, or the preferred task model is unavailable, the workflow runtime leaves the current/default model routing unchanged.
- Explicit monitor task models still win and bypass task-default substitution.
