# Workflow Snapshot Trigger Readiness

## Summary

The desktop daemon workflow snapshot now includes workflow-trigger readiness
for connector templates and configured connections.

## Behavior

- Connector rows include `can_trigger_workflow`.
- Connection rows include `can_trigger_workflow`.
- Readiness is computed from the same subscriber-manifest and connector-command
  checks used by native workflow creation.
- Existing connector setup hints, connection state, action slugs, and workflow
  run data are unchanged.

## Compatibility

The fields are additive. Older desktop clients can ignore them, while newer
clients can avoid staging invalid workflow triggers.
