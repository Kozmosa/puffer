# Contact Save Test Module Split

## Summary

Contact save regression tests now live in a dedicated
`daemon_contacts_save_tests.rs` module.

## Design

- Save normalization and saved-proposal pruning tests moved out of the large
  aggregate contacts test file.
- `daemon_contacts.rs` registers the new test module alongside the existing
  delete and aggregate contact tests.
- Runtime behavior is unchanged.

## Compatibility

This is a test-only structure change that keeps contact test files farther from
the Rust line-count contract.
