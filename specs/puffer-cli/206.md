# Contact Save Reopen Regression

## Summary

Contact save tests now cover reopening behavior by listing contacts through a
fresh config-path instance after a save.

## Design

- The save test module verifies that `contacts_save` writes `contacts.json` and
  `contacts_list` can read the saved contact after constructing fresh paths.
- The test keeps the runtime contract explicit without changing contact storage
  behavior.

## Compatibility

Saved contacts continue to live under the workspace runtime contacts store. The
new coverage guards against regressions where the UI appears empty after
reopening the Contacts screen.
