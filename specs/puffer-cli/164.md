# Gmail Browser Diagnostics

- Gmail browser subscriber startup now logs its topic, state directory, config,
  seen-cache, and auth-state file presence without printing account emails.
- Gmail inbox polling logs per-account hashed identifiers, DOM selector counts,
  timeout state, auth-required transitions, poll errors, and emitted row metadata.
- The inbox DOM script reports candidate, visible, and filtered row counts with
  a selector version so selector drift is visible in daemon logs.
- Daemon-owned Gmail setup logs discovery status, account counts, selection
  counts, saved state directory, and connection upsert outcome.
- Monitor creation and monitor history RPCs log connection state and history
  counts so an empty Tasks view can be distinguished from missing monitor runs.
