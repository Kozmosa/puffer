# Google Calendar event listing action

- Adds `list_events` to the `gcal-browser` connector action surface, with
  aliases such as `search_events`.
- The action reuses the browser-backed Calendar agenda extraction path and
  returns visible event rows with IDs, URLs, titles, timing text, and summaries.
- Inputs may filter by `query`, `title`, `event_id`, `url`, `when`, and
  `location`, with `limit` clamped to 1..50. `get_detail`, `accept`, and `deny`
  remain event-target actions and should receive an event ID, title, or URL.
