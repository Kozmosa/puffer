# Workflow Snapshot Connect Commands

## Summary

The desktop `workflow_list` RPC now includes deterministic connector setup
guidance alongside each connector template.

## Behavior

- Connector snapshot rows include `suggested_connection_slug`.
- Connector snapshot rows include `connect_command`, formatted as
  `/connect <connector_slug> <suggested_connection_slug>`.
- Existing workflow, run, connection, capability, and action-slug fields are
  unchanged.

## Compatibility

The extra fields are additive. Older desktop clients can ignore them, and the
daemon still degrades to empty connector arrays plus `connector_error` when the
subscription manager cannot provide connector context.
