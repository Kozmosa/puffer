# Verified Skill approval toggle

The daemon exposes `set_lambda_skill_approval` for workspace or user Lambda
Skill library manifests. The request identifies `libraryId`, `sourceKind`, and
`requireApproval`, rewrites the manifest's `require_approval` field, then
returns the same library snapshot used by the settings UI.

Library and recognized-skill DTOs include `requireApproval` so Desktop can show
and update the effective behavior without asking users to edit YAML.
