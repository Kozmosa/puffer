# Contact Inference Inclusion Prompt

The model-backed contact inference prompt now asks the model to propose as many
reasonable contacts as possible, up to the existing 30-contact cap.

The previous `When uncertain, skip the candidate.` instruction is removed.
Inference should favor inclusion when a candidate has plausible evidence of a
real person or useful relationship, while retaining the existing exclusions for
spam, bots, mailing lists, sales outreach, transactional notifications, and
one-off low-value interactions.
