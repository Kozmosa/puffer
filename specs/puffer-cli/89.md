# Verified Skills Folder Removal And Path De-Dupe

The desktop daemon now exposes `remove_lambda_skill_library` so the Settings UI
can delete a Verified Skills folder manifest instead of pretending a disabled
folder was removed locally.

Folder imports are path-aware. If a newly selected folder contains an existing
smaller Verified Skills folder, the smaller manifest is removed before the
larger one is saved. If a selected folder is already contained by an existing
larger folder, the daemon returns the current snapshot without writing a
duplicate manifest. Snapshots also hide nested duplicate manifests that already
exist on disk.

Validation:
- `cargo test -p puffer-cli desktop_lambda_skill -- --nocapture`
