# Gmail Browser Action-Only Setup Step

- The Gmail browser connector sign-in wait state sends an action-only choice question with no fake `Done` or `Cancel setup` options.
- Continuing the setup only acknowledges that the user finished browser interaction; cancellation remains handled by the connector setup modal cancelling the turn.
- Account selection remains a multi-select question because it captures real Gmail account choices and the optional sign-in-another-account action.
