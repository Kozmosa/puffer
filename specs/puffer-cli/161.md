# Monitor Ignore Test Module Split

## Summary

The monitor-task ignore helper tests now live in a sibling test module.

## Contract

`daemon_workflows/monitor_task_ignore.rs` remains the production implementation,
while `daemon_workflows/monitor_task_ignore_tests.rs` contains the existing unit
coverage. The split is structural only and keeps the production file below the
repository line cap.

## Compatibility

No runtime behavior or JSON contract changes. Existing tests continue to cover
ignore marking, safe filter derivation, and prior-filter synchronization.
