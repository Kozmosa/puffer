# Serve Connector Surface Without HTTP Ingress

## Change

`puffer-cli` no longer builds or dispatches the HTTP ingress connector.

## Contract

- The workspace member and optional dependency for the removed connector crate
  are gone.
- The `connector-webhook` feature is removed.
- `connectors-all` now enables Telegram, Discord, Slack, Matrix, and email.
- `puffer serve` treats `webhook` as an unknown platform instead of a known
  but disabled connector.

## Compatibility

Existing bot and email connector feature gates are unchanged. A stale
`[connectors.webhook]` table in user config is ignored with the normal unknown
platform diagnostic.
