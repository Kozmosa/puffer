# Fast Contacts Preview Lists

The `contacts_list` and `contacts_search` daemon RPCs now return lightweight
connector candidate previews.

- Bulk list/search collection ranks candidates without attaching raw connector
  context payloads.
- Bulk list/search does not invoke live connector contact commands, which can
  legitimately block up to the connector command timeout.
- Full context remains available through `contacts_context`; inference now uses
  only prompt-sized cached snippets.
- The Contacts desktop screen can load candidate rows from large Telegram
  diagnostic files without waiting on per-message context sorting and payload
  cloning.
