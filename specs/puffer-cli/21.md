# Anthropic Daemon Turn Smoke

The daemon process smoke suite covers the Claude desktop alias and canonical
Anthropic turn path without relying on the desktop Playwright harness. The test
starts a local Anthropic-compatible mock, overrides the workspace provider
resource, creates a session through `providerId=claude`, and runs a turn without
provider overrides so persisted session routing selects `anthropic`.

The mock implements `/v1/models` and streaming `/v1/messages`, captures the
request body, and verifies that the user prompt and selected Claude model reach
the provider request. This pins the session creation, alias normalization, and
turn execution path that previously produced runtime-drop failures in desktop
flows.
