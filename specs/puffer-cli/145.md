# Desktop Secret Settings API

The daemon settings API exposes encrypted secret metadata and mutation methods
for the desktop settings page.

- `load_settings_snapshot` includes `secrets` with store path, key source,
  Chrome-import support, and non-secret summaries.
- `save_secret` encrypts a manually entered value into the user vault and
  returns a refreshed settings snapshot.
- `delete_secret` removes a stored secret by id and returns a refreshed settings
  snapshot.
- `import_chrome_secrets` imports macOS Chrome saved credentials into the
  encrypted vault and returns `{ settings, report }`.
- Raw secret values are accepted only by the save/import paths and are not
  included in settings snapshots.
