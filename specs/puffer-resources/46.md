# New Relic Webhook Skill

Bundled resources now include `newrelic-webhook`.

## Skill Contract

- Teach agents to run `/connect newrelic-webhook <connection-name>`.
- Document the deterministic default
  `/connect newrelic-webhook newrelic-webhook`.
- Explain that the preset writes the generic webhook connector config and must
  be served with `puffer serve`.
- Describe New Relic webhook destination setup using a JSON POST destination
  attached to a workflow.
- Provide a recommended payload with New Relic issue fields including
  `issueId`, `issueTitle`, `priority`, `state`, `stateText`, `triggerEvent`,
  `issuePageUrl`, `incidentIds`, `entitiesData`, condition/policy
  accumulations, `nrAccountId`, and `workflowName`.

## Compatibility

The skill is documentation-only and does not add new resource loader behavior.
