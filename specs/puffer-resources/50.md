# Figma Webhook Skill

## Scope

Adds a bundled `figma-webhook` skill that documents deterministic setup and
runtime behavior for the Figma webhook preset.

## Design

- The skill routes users to `/connect figma-webhook <connection-name>` with
  the deterministic default `/connect figma-webhook figma-webhook`.
- It documents that the preset writes the generic `[connectors.webhook]`
  configuration and is served by `puffer serve`.
- It lists the supported Figma Webhooks V2 events and the conversation grouping
  keys used by the webhook normalizer.

## Compatibility

The skill is additive and follows the same resource model as the other webhook
connector skills.
