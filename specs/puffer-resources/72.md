# Gmail Browser Skill Guidance

## Summary

Add an explicit `gmail-browser` skill and tighten ConnectorAct guidance so
agents use the Gmail Browser action names that the connector actually exposes.

## Design

- `resources/skills/gmail-browser/SKILL.md` documents `/connect gmail-browser`
  setup, the global Puffer browser profile, account selection, and the
  supported ConnectorAct action names.
- The skill explicitly says not to use `read_messages` for Gmail Browser;
  agents should use `list_emails` or `search_emails` for reads.
- The generic ConnectorAct tool description repeats that distinction so agents
  do not borrow Slack/Lark `read_messages` terminology for email connectors.

## Compatibility

No connector action alias is added. Runtime validation continues to reject
unsupported Gmail Browser action names.
