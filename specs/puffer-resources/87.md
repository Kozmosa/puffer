# Image Provider Catalog Recheck

## Summary

Provider image resources have been rechecked against current official image
generation docs and live gateway metadata.

## Catalog Contract

- OpenAI now includes `chatgpt-image-latest` with the same Images API select
  parameters as the GPT Image 1.x generation models. DALL-E 2 and DALL-E 3
  remain excluded because their API shutdown date has passed.
- BytePlus now includes `seedream-4-0-250828` alongside Seedream 5.0 Lite and
  Seedream 4.5. Seedream 4.0 and 4.5 keep only `size=2K` in the current select
  descriptor and do not declare `output_format`.
- MiniMax, MiniMax CN, Zhipu, xAI, OpenRouter, and Vercel AI Gateway remain
  unchanged after the documentation and live model-list recheck.

## Compatibility

The OpenAI and BytePlus additions expand selectable image model coverage without
changing request routing. BytePlus avoids the backend-unsupported
`output_format` field for all non-5.0-lite Seedream requests.
