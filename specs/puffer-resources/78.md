# TaskCreate Durable Task Prompt

## Summary

`TaskCreate` resource guidance now describes durable workflow task creation
instead of ordinary current-session progress tracking.

## Behavior

- Normal agents are directed to use `TodoWrite` for progress checklists.
- Non-monitor `TaskCreate` calls are documented as requiring explicit user
  intent in the surrounding prompt.
- Monitor triage guidance remains supported through `_monitor` or
  `monitor_connection` metadata.

## Compatibility

The schema is unchanged. Existing monitor and non-monitor tool callers continue
to work; the behavioral change is in model-facing guidance.
