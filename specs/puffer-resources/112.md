# puffer-resources 112: WorldRouter image catalog completion

WorldRouter image resources now include all documented generation models from
the public WorldRouter quickstart and model catalog:

- `gpt-image-2` on the OpenAI-compatible `/images/generations` route;
- `gemini-2.5-flash-image` on Gemini native `generateContent`;
- `gemini-3-pro-image-preview` on Gemini native `generateContent`;
- `gemini-3.1-flash-image-preview` on Gemini native `generateContent`.

`gpt-image-2` keeps the provider-level `images_json` execution. Gemini models
override execution with `gemini_generate_content`,
`https://inference-api.worldrouter.ai`, and
`/v1beta/models/{model}:generateContent`. The explicit host-level base URL
avoids joining the provider's OpenAI-compatible `/v1` base URL with the native
`/v1beta` route.

The Gemini parameter surface is conservative: only documented
`aspectRatio`/`imageSize` fields are exposed, and `gemini-2.5-flash-image`
does not send `imageSize`.
