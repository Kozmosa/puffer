# AskUserQuestion image preview guidance

The AskUserQuestion tool resource now documents markdown image support for
question text and option previews. Tool callers can provide QR codes or other
inspection images with standard markdown image syntax, preferably as generated
`data:image/*` URIs when the image is produced locally.

The guidance keeps preview use scoped to concrete artifacts that affect the
user's answer and preserves the existing single-select preview constraint.
