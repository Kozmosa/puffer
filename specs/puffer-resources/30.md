# Workflow action examples

- `WorkflowCreate` and `WorkflowValidate` tool descriptions now show the
  canonical `file_append` action for saving matching messages to `/tmp`.
- The descriptions mention that step shorthand is accepted for compatibility,
  but canonical action YAML remains the preferred model-visible contract.
- `WorkflowCreate` no longer uses top-level JSON Schema combinators; runtime
  validation still enforces that callers provide exactly one of `action` or
  `yaml_action`.
