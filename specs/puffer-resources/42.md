# PagerDuty Webhook Skill Resource

## Change

Bundled resources now include `resources/skills/pagerduty-webhook/SKILL.md`.

## Skill Contract

The skill documents:

- deterministic setup through `/connect pagerduty-webhook <connection-name>`
- default command `/connect pagerduty-webhook pagerduty-webhook`
- default serve path `/pagerduty`
- `.puffer/connectors.toml` writing through `[connectors.webhook]`
- `puffer serve` as the runtime entrypoint
- PagerDuty Generic Webhooks (v3) setup notes
- current lack of `x-pagerduty-signature` verification

## Compatibility

The bundled skill count increases by one. Tmux help snapshots and resource
loader tests must continue to parse all bundled skill markdown.
