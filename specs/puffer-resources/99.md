# VideoGeneration Scalar Parameter Schema

Date: 2026-06-09

## Summary

The bundled `VideoGeneration` tool resource now declares parameter overrides as
scalar values instead of string-only values.

## Behavior

- `parameters` accepts string, number, and boolean values.
- Non-scalar values remain outside the declared tool schema.
- The runtime still normalizes accepted scalars into the existing string
  parameter map before provider-specific adapters translate wire types.

## Compatibility

This matches common tool-calling model output for settings like
`duration: 5` while preserving the text-to-video-only surface.
