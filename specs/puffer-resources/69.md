# Bundled Ingress Skills Removed

## Change

Bundled skills for the removed HTTP ingress connector and its service presets
are deleted from `resources/skills`.

## Contract

- Skill discovery no longer advertises the generic ingress connector skill.
- Service-specific setup skills for GitHub, GitLab, Jira, Linear, Stripe, and
  related presets are absent from bundled resources.
- User and workspace skill overlays remain supported by the existing resource
  loader.

## Compatibility

This only changes bundled resource availability. User-provided skills with the
same names may still be loaded from higher-precedence resource roots.
