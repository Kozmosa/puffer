# Linear Webhook Skill Resource

## Summary

Bundled resources include a `linear-webhook` skill for Linear webhook setup
guidance.

## Behavior

- The skill documents `/connect linear-webhook <connection>` as the setup
  command.
- It explains that the connector is a Linear-specific preset over the generic
  HTTP webhook transport.
- It documents the workspace config shape under `[connectors.webhook]`, the
  default `/linear` path, and the need to run `puffer serve`.
- It describes the Linear headers and payload fields the webhook connector
  normalizes for agents.
- It states that the connector is setup-only until webhooks expose durable
  workflow trigger events.

## Compatibility

The new skill is additive. Existing skills, resource provenance loading, and
connector setup resources keep their prior behavior.
