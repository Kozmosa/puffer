# Browser skill alias simplification

## Purpose

Keep Browser skill instructions aligned with automatic Bash helper injection.

## Design

The Browser skill no longer explains how to print or install shell aliases. It
states that agents should run Browser commands as `browser ...` inside Bash and
then documents the Browser CLI command surface.

## Contract

The skill is a Browser usage reference, not a shell setup guide. Browser remains
an internal CLI-only tool and must not be described as a model tool call.

## Verification

Resource parsing tests continue to load the Browser skill frontmatter and body.
