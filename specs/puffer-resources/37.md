# Stripe Webhook Skill Resource

The bundled `stripe-webhook` skill documents the deterministic
`/connect stripe-webhook <connection>` setup flow, the default `/stripe` path,
and Stripe dashboard webhook configuration.

## Behavior

- The skill routes payment, checkout, invoice, subscription, and dispute
  webhook requests to the Stripe preset.
- It documents that the preset writes `[connectors.webhook]` and runs through
  `puffer serve`.
- It lists the Stripe Event object fields Puffer recognizes.
- It describes stable conversation grouping by account, object type, and object
  id.

## Compatibility

The skill is additive and does not change existing bundled skill names or
slash-command aliases.
