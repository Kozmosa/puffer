# Telegram skill guidance for local command

- The bundled Telegram skill now directs agents to use the local `/telegram`
  command for lookup workflows instead of routing `/telegram` through a prompt
  command.
- The examples cover `search-peers`, `list-messages`, and fast
  `search-messages --context 0`.
- The Telegram internal tool metadata describes search context as previous
  messages before each hit, matching the faster subscriber behavior.
- Authentication guidance remains `/connect telegram-login <connection>` so
  choices and secrets continue to flow through `AskUserQuestion`.
