# puffer-resources 113: Short drama skill rename

The short drama orchestration skill is now named `short-drama-generation`.

## Resource Contract

- The skill resource path is `resources/skills/short-drama-generation/SKILL.md`.
- The frontmatter `name` is `short-drama-generation`.
- The resource keeps `requires-action: true` so a successful activation still
  requires follow-up tool work before the turn can complete.
- No `short-drama` alias or compatibility resource is retained.

## Compatibility

Callers must use `short-drama-generation` for explicit skill invocation. The
old `short-drama` name is intentionally removed.
