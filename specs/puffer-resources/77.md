# TaskCreate Monitor Task Threshold

## Summary

The `TaskCreate` tool guidance now documents the same strict monitor task
threshold used by monitor triage prompts.

## Behavior

- Monitor task creation is limited to explicit user requests, schedules,
  deadlines, or service notifications that clearly assign work or request
  review/approval.
- Greetings, acknowledgements, FYI/status-only updates, generic bot summaries,
  GitHub-style notifications, and repeated source updates are not tasks without
  an explicit request or schedule.
- Useful information alone is not a task; ambiguous monitor events should not
  create tasks.
- Monitor triage may inspect surrounding context for ambiguous events, but the
  context must confirm a real user action, request, or schedule.

## Compatibility

The tool schema is unchanged. This only changes model-facing guidance.
