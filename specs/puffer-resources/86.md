# BytePlus Seedream 4.5 Output Format Correction

## Summary

BytePlus Seedream image descriptors now preserve the documented difference
between Seedream 5.0 Lite and Seedream 4.5 request parameters.

## Catalog Contract

- `seedream-5-0-260128` keeps `size=2K` and `output_format=png|jpeg`.
- `seedream-4-5-251128` keeps `size=2K` and does not declare
  `output_format`.

The native BytePlus image endpoint remains `images_json` at
`/images/generations`.

## Compatibility

Desktop capability defaults for Seedream 4.5 no longer include an output
format selector, so new saved settings will not send a backend-unsupported
`output_format` field.
