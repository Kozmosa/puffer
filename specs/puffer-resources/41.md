# Sentry Webhook Skill Resource

## Change

Bundled resources now include `resources/skills/sentry-webhook/SKILL.md`.

## Skill Contract

The skill documents:

- deterministic setup through `/connect sentry-webhook <connection-name>`
- default command `/connect sentry-webhook sentry-webhook`
- default serve path `/sentry`
- `.puffer/connectors.toml` writing through `[connectors.webhook]`
- `puffer serve` as the runtime entrypoint
- Sentry Integration Platform and project service hook setup notes
- current lack of `Sentry-Hook-Signature` verification

## Compatibility

The bundled skill count increases by one. Tmux help snapshots and resource
loader tests must continue to parse all bundled skill markdown.
