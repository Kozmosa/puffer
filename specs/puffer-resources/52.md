# Telegram skill sender-filter documentation

## Scope

The bundled Telegram internal tool schema and skill instructions document
sender-filtered message listing.

## Contract

- The Telegram tool schema allows `sender`, `from`, `from_user`, and
  `scan_limit` for `list_messages`.
- The Telegram skill describes `/telegram list-messages --from <sender>` and
  the bounded `--scan-limit` behavior for sparse senders.

## Compatibility

No skill invocation names change. Auth remains routed through `/connect
telegram-login <connection>`.
