# Seedance 2.0 Video Parameter Coverage

Date: 2026-06-09

## Scope

Bundled video provider resources declare the configurable Seedance 2.0 options
that Puffer can currently express through string enum media parameters.

## Provider Coverage

- BytePlus declares `dreamina-seedance-2-0-260128` with `duration`, `ratio`,
  and `resolution` options for text-to-video generation.
- Relaydance keeps the existing `doubao-seedance-2-0-720p` model id and expands
  its declared Seedance 2.0 options to the same duration, ratio, and resolution
  sets supported by the adapter request mapping.
- Relaydance uses `Seedance 2.0` as the display name because resolution is now
  selected through the `resolution` parameter.
- Boolean, numeric, URL, media-reference, callback, and object parameters stay
  undeclared until `MediaCapabilityParameter` supports typed controls.

## Governance

Only providers with implemented Puffer video execution adapters may declare
`media.video`. The current executable bundled providers are BytePlus and
Relaydance.
