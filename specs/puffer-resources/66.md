# Lambda Skill Bridge Prompt

Imported Lambda Skill content now tells the model to call `LambdaHostCall`
with formal `host_tool`, formal `args`, and target concrete `tool`, while
omitting `input` unless Puffer explicitly returned an exact retry payload.
Puffer materializes concrete input from the verified host catalogue contract.

This keeps model-facing skill instructions aligned with the harness-side
binding path and avoids teaching the model to hand-author Bash or other
concrete tool payloads.

Validation:
- `cargo test -p puffer-resources lambda_skill_library_imports_generated_external_skills -- --nocapture`
