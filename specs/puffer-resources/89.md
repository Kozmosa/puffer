# ImageGeneration Tool Output Directory

## Scope

The bundled `ImageGeneration` tool metadata should describe the current image
output directory contract.

## Change

- The `outputPath` schema description now states that paths are relative to
  `.puffer/media/images`.
- The field remains optional; omitted values still use
  `generated-<timestamp>.<ext>`.

## Compatibility

The resource metadata no longer documents `.puffer/workflows/images` as a valid
output root.
