# Telegram skill command wording

- The bundled Telegram skill no longer spells out a connector-specific Telegram
  slash command in active guidance.
- The skill continues to direct agents to authenticate through `/connect
  telegram-login <connection>` and to use the hidden internal `telegram ...` CLI
  for peer lookup and message search.
- This keeps Telegram behavior available to workflows while avoiding a
  user-facing slash-command affordance for a single connector.
