# Shopify Webhook Skill Resource

## Summary

Adds the bundled `shopify-webhook` skill so agents and users can discover the
deterministic setup path for Shopify webhook deliveries.

## Skill Contract

The skill documents:

- `/connect shopify-webhook <connection-name>`
- deterministic default `/connect shopify-webhook shopify-webhook`
- default serve path `/shopify`
- `[connectors.webhook]` storage because the preset is served by the generic
  webhook connector
- `puffer serve` as the runtime command
- Shopify HTTPS JSON webhook subscriptions
- `X-Shopify-Topic`, `X-Shopify-Shop-Domain`, `X-Shopify-Webhook-Id`,
  `X-Shopify-Triggered-At`, and `X-Shopify-Event-Id` behavior

## Compatibility

Bundled skill count increases by one. The tmux help wide snapshot is updated so
resource counts remain deterministic.
