# Verified Skill approval preference

Lambda Skill library manifests may set `require_approval: true` to keep normal
user approval prompts for concrete tool calls that pass the verified gate.

The field defaults to `false`. A loaded verified skill therefore carries
`verification.require_approval = false` unless the external library manifest
opts into the additional prompt. The loader also accepts the camel-case
`requireApproval` spelling used by Desktop RPC payloads.
