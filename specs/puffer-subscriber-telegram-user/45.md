# Telegram Bot Contact Metadata

- Telegram user subscriber event payloads and diagnostics now include
  `chat_is_bot` and `sender_is_bot` booleans when Telegram identifies the chat or
  sender as a bot user.
- Contact inference and ranking consumers use these flags to avoid listing bots as
  standard Telegram contacts.
- The fields are additive JSON payload metadata and do not change delivery,
  acknowledgement, subscription matching, or persisted session behavior.
