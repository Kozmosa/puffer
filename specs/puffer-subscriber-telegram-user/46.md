# Telegram Full Names in Message Metadata

## Design

Telegram message events and message diagnostics now use the full Telegram user
display name for user chats and senders. Group and channel names keep using
their chat title.

## Logic

The subscriber calls `User::full_name()` when a `Chat::User` supplies the chat
title or sender name. Bot detection also treats Telegram usernames ending in
`bot` as bot users, and bot messages are omitted from the durable
`message-diagnostics.ndjson` feed used for contact discovery.

## Compatibility

Payload field names stay the same: `chat_title`, `sender_name`,
`chat_is_bot`, and `sender_is_bot`. Existing consumers receive more complete
names without a schema migration, while live Telegram event delivery remains
unchanged.
