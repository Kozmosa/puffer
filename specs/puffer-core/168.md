# SpotifyAction Lambda Bridge

Puffer now exposes `SpotifyAction` as a typed concrete tool for verified Lambda
Skill contracts that map directly onto Spotify Web API operations. The runtime
supports playback mutation, device transfer, queue add, playlist create/add, and
library list/save/remove/contains using the exact concrete input admitted by the
Lambda host bridge.

The tool reads `PUFFER_SPOTIFY_ACCESS_TOKEN` or `SPOTIFY_ACCESS_TOKEN`, sends
only HTTPS requests to `api.spotify.com`, and fails closed for unsupported
resource/action pairs or malformed URI/id inputs. It is listed in the daemon
Lambda concrete-tool allowlist so Desktop imports can accept host catalogues
that bind formal Spotify host tools to this concrete adapter.
