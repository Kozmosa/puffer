# Connect And Tool Executor Module Splits

## Summary

Large core helper files were split to satisfy repository contract checks.

## Contract

`command_helpers/connect.rs` keeps the `/connect` dispatcher and shared prompt
helpers. Lark CLI setup now lives in `command_helpers/connect/lark_cli.rs`.
`runtime/tool_executor.rs` keeps production execution logic while its pentest
scope tests live in `tool_executor_scope_tests.rs`.

## Compatibility

The `/connect` Lark, Slack, Telegram, email, and generic connector flows keep
their existing call paths and test coverage. Tool executor pentest-scope behavior
is unchanged.
