# Anthropic Stream Bridge

Puffer now exposes an `AnthropicStream` concrete tool for verified Lambda Skill
contracts that consume a previously opened Anthropic SSE stream.

The first action is `finalMessage`. It accepts raw SSE text, or an object with
`stdout`, `output`, `stream`, `text`, `body`, or `content`, and extracts the
assistant text from Anthropic `content_block_delta` events or a final message
JSON object.

The bridge is intentionally local: it does not issue authenticated HTTP
requests, spawn shell commands, or infer API credentials. Skills that need to
open a stream must still bind that operation to an explicit concrete HTTP or
shell call. Skills that only parse the returned stream can bind to this typed
bridge instead of `LambdaInternal`.
