# Stripe Webhook Connect Flow

`/connect stripe-webhook <connection>` routes to the serve-mode webhook
connector with deterministic setup questions.

## Behavior

- The flow asks for a bind address and URL path.
- Blank paths default to `/stripe`; relative paths are normalized with a leading
  slash.
- The workspace connector config is written as `[connectors.webhook]` because
  the preset uses the generic webhook listener.
- The generated config stores `display_name`, `bind_address`, `path`, and a
  `Stripe webhook ready.` welcome message.
- `/workflows connectors` search can resolve Stripe billing terms to
  `stripe-webhook`.
- The connector row reports `runtime=serve`, `connect=/connect stripe-webhook
  stripe-webhook`, and `[no-trigger]`.

## Compatibility

The preset is additive and does not change provider auth, connector protocol
subscribers, or existing generic webhook behavior.
