# Video Workflow Seedance Fixture Options

Date: 2026-06-09

## Scope

Core video workflow tests use Seedance 2.0 parameter fixtures aligned with the
bundled provider resources.

## Behavior

- Relaydance workflow fixtures include duration values from 4 through 15
  seconds.
- Resolution fixtures include `480p`, `720p`, and `1080p`.
- Aspect ratio fixtures include `16:9`, `4:3`, `1:1`, `3:4`, `9:16`, `21:9`,
  and `adaptive`.

## Compatibility

Runtime request mapping is unchanged; the fixture still verifies that selected
settings are passed through the existing Relaydance adapter parameter mapping.
