# ComfyUI Verified Skill Bridge

## Scope

Adds a narrow `ComfyUiAction` runtime bridge for verified Lambda Skill
catalogues. The initial action is `configureCloud`, used by the translated
ComfyUI skill's cloud setup path.

## Contract

`configureCloud` requires:

- a setup choice value whose recorded user-facing text contains `cloud`;
- an API-key source report with `configured=true` and a `sources` array;
- a live `COMFY_CLOUD_API_KEY` value available from the reported environment
  variable or dotenv source.

The bridge refuses raw API-key strings. It probes the fixed Comfy Cloud host
with the resolved key, returns `https://cloud.comfy.org`, and reports
`credentials_redacted=true`. It does not persist the secret or include it in
tool output.

## Compatibility

The bridge is model-invocable only through generated verified-skill contracts.
Generic ComfyUI workflow execution still uses the existing script and REST
contracts; this change only replaces the prior effectful `LambdaInternal`
placeholder for cloud configuration.
