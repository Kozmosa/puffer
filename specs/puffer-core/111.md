# Telegram message sender filter command

## Scope

The local `/telegram list-messages` command can request recent messages from a
single sender inside a Telegram peer.

## Design

- `/telegram list-messages` accepts `--from`, `--sender`, or `--from-user`.
- The parser sends the normalized request as `sender` to the consolidated
  Telegram workflow tool.
- `--scan-limit` bounds how many recent history messages the subscriber may
  inspect while collecting sender-filtered matches.
- Succinct message-list output reports the sender filter, scanned count, scan
  limit, scan-limit exhaustion, and the older-page cursor.

## Compatibility

Existing list-message calls without a sender filter keep the same default limit,
cursor behavior, and succinct output shape. Search-message behavior is
unchanged.
