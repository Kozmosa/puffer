# Static Video Capability Flow Coverage

Date: 2026-06-09

## Scope

Core media capability tests now cover bundled static Seedance video descriptors
flowing through `list_exact_media_capabilities_with_cache`.

## Behavior

- Connected BytePlus and Relaydance auth stores expose all bundled static video
  capabilities as `available`.
- Capability output remains descriptor-driven with `source = static` and
  `operation = generate`.
- BytePlus fast resolution limits and Relaydance resolution-specific
  single-value parameters are preserved in the capability view.
- BytePlus and Relaydance request-body tests assert newly cataloged model ids
  are serialized unchanged by the existing video adapters.

## Compatibility

No resolver or runtime discovery behavior changed. Video capabilities still
come only from provider `media.video.models`.
