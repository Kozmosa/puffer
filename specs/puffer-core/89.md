# Linear Webhook Connect Flow

## Summary

`/connect` and `/workflows connectors` understand the `linear-webhook` preset.

## Behavior

- `/connect linear-webhook <connection>` dispatches to a deterministic
  serve-mode setup flow instead of the generic connector path.
- The flow asks for bind address and URL path through `AskUserQuestion` input
  questions, normalizes blank or relative paths to `/linear`, and writes a
  `[connectors.webhook]` workspace config.
- Connector search resolves terms such as `linear issue` to
  `linear-webhook`.
- `/workflows connectors linear issue` shows the preset as setup-only with
  `runtime=serve`, `connect=/connect linear-webhook linear-webhook`, and
  `no-trigger` capability.

## Compatibility

The change is additive. Existing `/connect webhook`, `/connect github-webhook`,
and workflow trigger validation behavior remain unchanged.
