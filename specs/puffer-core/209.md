# Telegram Connect Diagnostics

## Change

Telegram personal-account setup now surfaces actionable diagnostics when the
local subscriber manifest or subscriber process cannot be reached.

## Contract

- Missing `telegram-user` manifests report every searched `manifest.toml` path.
- The error names `PUFFER_BUILTIN_RESOURCES_DIR` as the bundled-resource escape
  hatch for daemon or packaged-app layouts.
- Unreadable manifests add `load Telegram subscriber manifest <path>` context.
- Telegram subscriber startup writes `[telegram-connect]` stderr lines with the
  connection slug, subscriber id, cwd, bundled resources dir, and state dir.
- Startup failure errors retain a `start Telegram subscriber` context string.

## Compatibility

No Telegram login protocol, credentials, session storage, or connector action
shape changes. Logs intentionally omit phone numbers, login codes, passwords,
and Telegram message content.
