# Recall Auxiliary Context

- `Recall` is a workflow tool registered as `runtime:workflow:recall` through `resources/tools/recall.yaml`.
- The handler lives in `crates/puffer-core/runtime/claude_tools/workflow/recall.rs` and ranks the current `AppState` transcript with a dependency-free lexical score.
- Tool input is `{ query: string, k?: number }`; `k` defaults to 5 and is capped at 20.
- Empty queries are rejected, and the current Recall tool call is excluded so the tool does not echo itself.
- Results replay the best matching prior transcript items with actor, item kind, score, and verbatim content so compacted or older session detail can be recovered during a turn.
- The first implementation is offline and lexical only. A later semantic index can use local providers such as MiniCPM5 without changing the public tool contract.
