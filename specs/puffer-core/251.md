# Image Generation Media Image Root

## Scope

The workflow `ImageGeneration` tool stores generated image files under the same
workspace image media directory used by chat image generation.

## Change

- The workflow image output root is now `.puffer/media/images/`.
- Default filenames remain `generated-<timestamp>.<ext>`.
- Explicit `outputPath` values remain relative to the image output root and
  must still be safe relative paths.

## Compatibility

No fallback to `.puffer/workflows/images/` is retained. Callers that pass
explicit paths should treat them as image-root-relative paths under
`.puffer/media/images/`.
