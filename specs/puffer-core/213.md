# Google Calendar Connect Helper

The deterministic `/connect` helper recognizes `gcal-browser` outside the
desktop daemon setup path.

- `/connect gcal-browser [connection]` asks for Google account email addresses
  through the standard question flow.
- The helper writes `gcal-browser-accounts/<connection>/config.toml` with the
  current workspace root and normalized account list.
- It creates or updates the matching authenticated connection record using the
  `gcal-browser` connector slug.
- The daemon path remains richer and browser-native; this helper is a fallback
  for non-desktop contexts that still need deterministic setup.
