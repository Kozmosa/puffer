# Shopify Webhook Connect UX

## Summary

Adds `shopify-webhook` to deterministic `/connect` and connector search UX.
Users can search for Shopify order/product/customer terms in the TUI catalog and
run:

```text
/connect shopify-webhook shopify-webhook
```

## Design

The shared `connect_webhook_preset` dispatcher maps `shopify-webhook` to Shopify
display text, default path, and summary method.

Shopify uses:

- bind prompt: `What bind address should the Shopify webhook listen on?`
- path prompt: `What URL path should Shopify post webhook events to?`
- default path: `/shopify`
- welcome message: `Shopify webhook ready.`
- method: `Shopify event webhook`

## Compatibility

Existing webhook preset behavior is preserved for Asana, GitHub, GitLab, Jira,
Linear, Stripe, and Trello. The connector remains setup-only in `/workflows
connectors` and is searchable by Shopify, order, product, customer, inventory,
and webhook terms.
