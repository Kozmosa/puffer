# Figma Webhook Connect UX

## Scope

Adds `figma-webhook` to the deterministic `/connect` and workflow connector
search surfaces.

## Design

- `/connect figma-webhook <connection>` uses the generic webhook serve-mode
  writer and stores the connection under `[connectors.webhook]`.
- The preset defaults the webhook path to `/figma` and describes the setup
  method as `Figma webhook`.
- Connector search metadata marks the preset as `serve`, setup-only, and
  searchable by terms such as `figma`, `comment`, `library`, and `dev mode`.

## Compatibility

No existing connector slug changes. The preset follows the same serve-mode
contract as the other webhook presets and does not claim workflow trigger or
connector action capabilities.
