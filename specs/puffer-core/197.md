# Verified concrete tool approval

Verified Lambda Skills default to skipping the normal user approval prompt only
for the concrete tool call admitted through `LambdaHostCall`. The skip is
allowed when the pending bridge does not require approval and the next concrete
tool name plus canonicalized input exactly match the precompiled contract.

Explicit denial rules still deny the call. When `require_approval` is enabled,
the runtime preserves the existing ask flow. Filesystem path prompts follow the
same verified skip so exact admitted file tools do not pause again unless the
library opts into approval.

`/skills` and `/doctor` include whether verified tool approval is required or
skipped for each verified skill.
