# Connector Catalog Without HTTP Ingress

## Change

The core command surface no longer exposes HTTP ingress connector presets.

## Contract

- `/connect` dispatches deterministic serve setup only for Telegram, Discord,
  and Matrix bot connectors.
- Built-in connector search now reports ten connectors.
- Runtime hint filtering uses `serve` only for the remaining serve-mode bot
  connectors.
- Queries that previously matched GitHub, GitLab, Linear, Stripe, or other
  ingress presets now return no built-in connector match.

## Compatibility

Provider-backed and internal-tool connector flows are unchanged. Unknown
connector slugs still fall through to the generic connector path so workspace or
user-defined connector templates can continue to evolve independently.
