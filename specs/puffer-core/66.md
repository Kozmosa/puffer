# Workflow Connector List Hints

## Summary

`ConnectorList` now returns actionable connection hints for every connector row.
This aligns the model-facing workflow tool with the `/workflows connectors`
summary and desktop workflow snapshot.

## Contract

- Each connector row includes `suggested_connection_slug`.
- Each connector row includes `connect_command`.
- `connect_command` is formatted as
  `/connect <connector_slug> <suggested_connection_slug>`.
- Suggested slugs come from `puffer_subscriptions::suggested_connection_slug`,
  so built-in defaults stay shared across CLI, TUI, desktop, and tool output.

## Compatibility

Existing connector fields are unchanged. Clients that ignore unknown fields keep
working, while agents and UIs can use the new hints to create deterministic
connections without inventing connector-specific names.
