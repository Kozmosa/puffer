# Telegram QR approval preview

The deterministic `/connect telegram-login` QR path now renders the exported
`tg://login` URL as a markdown image preview. The core command helper generates
an SVG QR code in process and embeds the result as a base64
`data:image/svg+xml` URI.

The raw login URL is still included below the image so the flow remains usable
when the user wants to inspect or copy the underlying Telegram URL.
