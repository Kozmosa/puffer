# Lambda Refinement Symbol Equality

## Summary

Puffer's Lambda Skill runtime treats unquoted identifiers on the right side of
`=`/`==`/`!=` refinements as enum-like symbols. JSON host arguments commonly
carry those witnesses as strings, for example `TrelloCred{sec = secret}` with a
formal value of `"secret"`.

## Runtime Contract

- Numeric comparisons are unchanged and still require numeric operands.
- Equality and inequality now compare JSON strings with symbol literals by
  symbol name.
- Other value comparisons retain exact typed equality.

This keeps generated catalogue readiness honest for `sec = secret` and similar
proof-only refinements without weakening unsupported predicate handling.
