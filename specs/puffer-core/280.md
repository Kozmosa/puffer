# puffer-core 280: Short drama generation skill fixtures

Runtime skill-action obligation tests now use `short-drama-generation` as the
short drama skill name.

## Runtime Contract

The obligation guard remains generic. It still keys behavior from the resolved
skill resource's `requires_action` flag and does not special-case any short
drama skill name.

## Compatibility

No runtime alias is added for `short-drama`. Tests and fixtures use only the new
`short-drama-generation` name.
