# Lambda Skill Self-Validator Predicates

## Summary

The Lambda Skill gate now treats the small set of generated self-validator
predicates as semantic predicates instead of fact-only predicates.

## Runtime Contract

These predicates are checked directly at host-call admission:

- `known_app` for supported agent deep-link targets.
- `valid_provider` for CodexBar model-usage providers.
- `valid_precision` and `valid_backend` for Accelerate configuration choices.
- `plan_path` for `.hermes/plans/*.md` plan output paths.
- `codec_license_ok` for TouchDesigner codecs that do not require a commercial
  license.

## Compatibility

The change prevents validator host tools from requiring facts that only those
same tools could produce after admission. Unsupported values still fail closed.
