# Discord Verified Search Bridge

## Scope

`DiscordAction` now supports the `searchMessages` action for verified Lambda
Skill contracts that need authenticated Discord message lookup. The bridge uses
the Discord Bot API with Puffer-owned bot-token lookup and does not expose raw
HTTP construction to the model.

## Contract

The concrete input binds `service`, `guildId`, `channelId` or `channelIds`,
`query`, and `limit`. Puffer validates Discord snowflakes, rejects unsupported
services, discovers guild text channels when a guild is supplied without
explicit channels, and searches recent channel history case-insensitively.

## Compatibility

Existing `sendEmbeds` and `readMessages` inputs are unchanged. Search requires
`PUFFER_DISCORD_BOT_TOKEN` or `DISCORD_BOT_TOKEN` and the bot's normal Discord
permissions to read channel history.
