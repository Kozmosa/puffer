# Telegram message-list command

- `SubscriberCommand` now includes `TelegramListMessages`.
- The command carries `peer`, optional `limit`, optional `before_id`, and a
  succinct-output flag.
- `before_id` is an exclusive Telegram message id cursor for fetching older
  pages from the same peer.
- The command is additive and preserves the existing JSON tagged-command
  protocol. Subscribers that do not handle Telegram commands continue to ignore
  it explicitly.
