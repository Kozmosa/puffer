# Connector Removed

## Change

The HTTP ingress connector crate is removed from the Cargo workspace.

## Contract

- No workspace crate, feature gate, or serve dispatcher remains for this
  connector.
- The shared connector catalog no longer exports service presets backed by this
  transport.
- Bundled skills for the connector and its service presets are removed.

## Compatibility

Historical specs remain append-only for auditability. Runtime behavior now
treats the old connector slug as unknown unless a user supplies their own
out-of-tree implementation.
