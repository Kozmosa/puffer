# puffer-connector-core update 4

Connector runtime session writes preserve tool invocation metadata.

## Design

Connector turns copy non-null `ToolInvocation.metadata` into append-only
`TranscriptEvent::ToolInvocation` records.

## Contracts

- Connector-visible messages and tool output text are unchanged.
- Missing metadata remains represented as an omitted event field.
- Actor and subject attribution are preserved exactly as before.

## Compatibility

External connector sessions can retain Lambda Skill trace annotations while
older transcripts without metadata continue to load.
