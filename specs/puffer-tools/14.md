# Media Internal Tool Visibility

The tool registry treats media generation as internal-only.

- `ToolRegistry::from_resources` registers `ImageGeneration` and
  `VideoGeneration` through the internal definition table, including their CLI
  aliases.
- `ToolRegistry::definition("ImageGeneration")` and
  `ToolRegistry::definition("VideoGeneration")` return none, so provider tool
  definition export does not include media generation.
- Internal shell helper generation includes `imagegen` and `videogen`, each
  forwarding to the corresponding `puffer internal-tool` subcommand.
